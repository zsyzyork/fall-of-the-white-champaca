# 第三回 落花廡故友嚐舊味，鳴鳳盃新生諷今時

话说白宛莹正悻悻折返，校道旁突然间闪出个人。宛莹见此人正是孟驰云，便佯装不理、加速前行。临走到教学楼下了，也不见驰云追来。她不禁心中寥萧，回头一看，却见驰云轻手轻脚地紧跟在脑后，便又气又笑道：“都几岁人了，还这么幼稚。”驰云笑道：“本来没打算吓妳的；只是见你耷拉着头，一路没声没气的走着，就想给你提提神。”宛莹责怪道：“非要这时才打招呼，上午在课室就不理我。”驰云解释道：“一年多不见，你又换了发型。高上治没说是妳，我都不敢认了。”
宛莹听了，直在原地跺脚，抱怨道：“我就知道是这该死的头发！”驰云回头仔细看了看她的脸，笑道：“剪短了，反而更有灵气些。”宛莹听了，心里略暖了些，却又将信将疑，便说：“哟，你是几时学会安慰人的？”自己也端详了驰云半刻，笑问道：“你是不是瘦了？”
驰云并不接话，只问：“妳怎么也来咱们学校了？吃了没有？怎么没去午休？”宛莹闭眼摇头，无奈笑叹到：“还是这样。你叫我回答哪个是好？”，又说“你们那高老师偏不让我走。刚去餐厅和宿舍，都已经关门了；然后就被你吓了一跳，然现在你又开始盘查我为什么到一中上学，可见我是讨人嫌的了。”孟驰云笑道：“哪里讨人嫌了。妳来了最好。咱们都没吃饭，先找点东西垫垫肚子再说。”宛莹道：“你自己也不吃饭午休，还问人呢。”
孟驰云皱眉道：“这种鬼天气，食堂也不卖几个凉菜，连糖水都是烫的，还没吃完就一身的汗──谁愿意去？我反馈给学生会，学生会还振振有辞，说什么热天喝冷饮不养生──养生书还说‘饭后百步走，活到九十九’呢，学校今日却让我们先徒步十公里再开饭；可见是强词夺理。宿舍也一个鬼样，连冷气都没有。八个人挤在里面，又热又湿，有什么意思。”
白宛莹笑道：“今天的糖水可不是热的，”又把官佩珩禀报班长的话一一转述。孟驰云听了，也只有点头称许：“难为她想得比别人都周全。只是这种天气，常温糖水也嫌热──咱们要不要吃点冰的？”不等宛莹回话，驰云又领着她掉头前往福利社，进去买了两碗豆花，一对糯米糍冰淇淋，再折回草坪，到男一舍底楼的外廊里坐下。宛莹看见豆花包装上的水鹿标识，便知是哪一家的──那家豆腐屋的总店就在宛莹的国中旁边。他们家豆花像双皮奶，是放在纸碗里蒸熟的；临尾还在表面浇上蜜豆、绿豆沙、椰汁西米等配料，别具一格，深得学生喜爱。
宛莹揭开碗盖，只见买的是绿豆沙口味的豆花。驰云却不着急吃，而是又把冰淇淋大福堆在豆花上，便笑道：“吃个甜品，还搞这么多花神。”驰云却早已跳到一株粗枝大叶的矮树下，撷了朵黄白相间的花，用草坪喷头边略洗了一洗，折下花瓣，也洒进纸碗里。宛莹忙阻止道：“花也能乱吃？”
驰云笑道：“不妨事。这是棉栀子，就是在地人叫做鸡蛋花的。”宛莹又问：“你就不怕农药？”驰云说：“你没看见，扫地阿姨得闲的时候，自己也把这树的花摘了去煲糖水，可见她们并不打药。这里原是热带雨林，草木本来就茂盛；这棵棉栀子，还有那边几棵白兰，说是有一百多岁了，比一中还长寿。妳们宿舍楼前还有杨桃、莲雾、波罗蜜，结的果比外面卖的还要好。听闻前年这里原本还有二十几棵高产荔枝树，刚好交给温良恭俭各班认养；开春的时候，荔枝多得大家吃到吐。可惜去年为了修个游泳池，好端端的树竟全部被铲了──也不知学校是怎么想的。”
宛莹不理他，舀起一勺豆花尝了，浑身稍觉清新凉爽；又有那鸡蛋花在之中沁出缕缕别致甜香。驰云道：“这是古早关西一带洋商买办人家的吃法，本来是绿豆沙加半勺甜糯粥、一匙桂花酱。现今没有这些，只能就地取材，取个意思罢了。妳别吃豆花，也试试这面上的冰淇淋。”
宛莹尝了一口，其实就是寻常的冰淇淋大福味道；跟豆花一起吃进嘴里，也只是口感微妙了些许而已。只是她错过了午饭，原以为要饿肚子的，不想竟有甜点可吃，又有人可谈天。两人一路吃，一路说着话，宛莹竟渐渐忘掉了方才的烦心事，却挖苦驰云道：“这么刁钻，比念书还花心思了，亏你想得出来。”
孟驰云笑道：“本来这福利社我也少来，只是听室友说，这里租期马上就到了，下学期便不租给福利社，只许华洋集团进驻，变成高级超商。这手制豆花，怕是吃一次少一次了。”
“华洋超商？”宛莹惊问，“那里边卖的和果子、洋巧克力、零嘴泡面不都是原装进口的吗？这么奢侈的东西，哪个学生会买？”驰云道：“我原来也这样想，可我室友说得也有道理：咱们西江一中不比别处，许多学生都出身官商人家。这新店自然是为他们开的了。我们虽不至于吃不起，只是多使十倍冤枉钱，两三百块也未必买得到这古早味。”
白宛莹道：“这也奇怪。那些小姐少爷是从哪里来的，咱们班倒没见到几个。”驰云笑道：“温班的富家子弟最多了！只是家里世代做官经商，懂得韬光养晦，与那些暴发户到底有差。刚才妳說的官佩珩，爸爸就是议长；本身姓官，又是个官。还有那安若素不言不语的，其实她父亲也是个什么长。”宛莹惊叹道：“原来佩珩是官议长的女儿！你不说，我再也想不到的。”驰云道：“像她这样父母做官的好多呢；再不然就是有钱的；只是我听了也记不住。妳要好奇，问我室友去。”宛莹笑问：“你哪个室友？一个男生也这样八卦的？”驰云道：“他叫佟嘉宇，人品其实不错，只是特别喜欢钻研各家背景门第罢了。”宛莹又说：“我这种穷人家的女儿，去打听那些呢。”驰云笑道：“按白叔叔的头衔算，妳也同他们一类了。”宛莹嗔道：“又胡说。”
忽然几阵风来，摇枝撼叶。宛莹望去，只见那棵什么栀子树上所结乳黄花瓣翩翩而落，如梦似幻。驰云道：“以前从未安心坐在这里，今日才知原来男舍也有景致；可惜落花正对着牛吃牡丹的一群俗物，有谁会留意呢。”宛莹直指他身后的宿舍楼，羞道：“你也有打嘴的时候了。”驰云回头，看见骑楼檐下赫然一幅裱墙书法字帖，其上写着“落花廊”三个大字他虽羞赧万分，却转移话题道：“这‘廊’字又不对了。两面开敞才谓之廊，这里叫‘庑’才恰当。”
宛莹不答，扭头望过乱花，只见校道上一女学生正从宿舍走来；远远地看不真，唯见来人雪颜乌发、杏眼红唇，便悄拉孟驰云道：“那女生好美！走在花下，倒像迷路森林的白雪公主了。”驰云正笑说“又不是童话，哪来这样的人”，一眼望去，那边的人也回眸相顾；驰云忙低呼一声“不好！”白宛莹正迷惑，又听他抱怨：“碰到风纪巡查组了，而且今天偏偏是她！”宛莹犹笑道：“这世上竟有人是你怕的。”驰云却正色道：“我不会有事，只怕牵扯了妳。咱们快散吧。”宛莹虽不知所以，但看他神情慌张，又记起严禁男女独处的校规，也略感事态不小，于是连忙背过头去碎步跑开了。回头再看，孟驰云和那女生都已没了踪影。
眼见又有若干学生零零落落走过，宛莹便知午休已结束，便重返女舍办了入宿手续。宿管阿姨给配了钥匙，又将她带到房间门口。宛莹敲了门，开门迎接的恰是官佩珩；两人寝室相逢，不免又笑又抱，说了会闲话，便拉着手一齐走回教室。经过栀子花处，宛莹又想起那个吓到孟驰云的漂亮女生，便试探着问佩珩知不知此人姓甚名谁。佩珩思索好一阵，开玩笑道：“范围也太大了吧，有好多个呢。”
两人正经过良班门口，却撞见高上治在走廊上对孟驰云训话。宛莹看高老师笑吟吟的，却断续听得他说“校长就在那远远地看着”、“咱们班学生，更要严格要求自己”，又听驰云故作委屈地申辩“宿舍太热睡不着，我只是一个人下楼吃点冷饮；肯定是恭班风纪组的人看我们班不惯，乱给校长报信。”宛莹听见果然是中午事发，赶紧拉了佩珩溜进门去。
大部队休整片刻，便下广场操练。佩珩、晓娟仍来陪宛莹笑耍。佩珩记起宛莹方才的问话，便也问晓娟知否驰云所识恭班女生是谁。晓娟抢白道：“不是何诗言，还能有谁？国三时两个人风风雨雨了一整年，搞出多少事来，妳又不是──”等回过神来，却又望着二人点头笑道，“怎么，妳们谁对孟驰云有意思了？”不等佩珩反应，宛莹却脸红道：“我和他从小一个院子长大，不过是比别人熟些，但也没有什么。”望晓娟正欲调笑说“青梅竹马，难道还不算什么”，却幡然想起今早诋孟驰云毁太尽，怕得罪了宛莹，又怕她告诉驰云；于是自己讪笑着走了。颜槿华本来在看书的，这时却突然插嘴道：“他俩平时都不怎么睬咱们，咱们也没人知，全班恐怕就佟嘉宇还了解。妳可以去问他。”宛莹唯恐有人再提驰云，忙用别话支吾过去。
再说孟驰云闪到教室坐没多久，就被高上治抓了去训诫男女大防。驰云料定是何诗言通风报信，早有准备，少不得狡辩一番；却仍听了许多唠叨，下午训练也被高老师盯紧了三分。偏偏骆校长中途又威威赫赫地踱来巡视，应景地演讲什么“学生士兵无自由”、“服从就是天职”。驰云生来最讨厌这种无聊规矩、多余磨练，加以午间未食、头晕身热，百般不自在，一解散就诈病去宿舍躺着；傍晚有个温、恭两班的联谊会就更懒得去。
散训之后，高老师便上来叫自修众人也去联谊。宛莹只好随行下楼，在队列后边坐了。联谊内容无非军歌笑话，毫无意义；倒是两班女生在暗地里相互打量，宛莹自然少不得多瞥何诗言几眼。正出神中，突然身前一男生回头寒暄。白宛莹看他谦恭有礼，倒不似别人轻浮幼稚；与他攀谈了几句，才知此人正是佟嘉宇。二人言谈投机，竟越聊越近；半刻下来，这一中风俗、温班旧故，乃至谁同谁是远亲，谁又有什么遗传病，她已知了大半。
宛莹见周围众人都不理论，便再从他口中打探驰云与诗言之事──听他所言，诗言果然是驰云国中时的女友，两人好过一阵，后来却莫名其妙分了手。诗言本有些先天不足，加之以情伤太深，分手后某些生理状况也日益加重。不知情者只见驰云之失，更有一干宵小唯恐天下不乱，四处造谣说这是堕胎后遗症。有次诗言大出血报医，恰好是孟母的妇产科同事出急救；人还未送到医院，闲话就先传到了孟父耳里。孟父听了风言风语，自叹教子无方、尽失医德；不由分说便冲到国中部，拳脚并茂，把驰云揍了个左肾破裂；直到诗言的诊断结果出来，才还了两人清白。冤打案后，驰云诗言都心存芥蒂，相互躲避至今。诗言国中时文理偏科，并未分到直升班，原本也曾发誓努力考取温班与驰云相守；分手后分班考试果然考取了全级第一，却无论谁都劝不动她回温班了。
宛莹这才知原来骆校长口中的才女就是此人；又略略记得父亲之前说驰云今夏受伤住院，万万想不到是因为这段隐情，真真可笑可叹；再细瞄诗言，果然气质如清洁玉树、傲雪莲花；便觉两人实在天造地设、般配登对。嘉宇见她恍惚，只道她乏了，便不去唐突。宛莹回神自觉失礼，又不便主动搭讪示好，也就闲闲地坐到联谊结束。

自此之後，軍訓期間白宛瑩日日到校，與溫班眾學生也日漸熟絡，不似原來那般恐懼生分了。轉眼已到八月底的入宿時分。宛瑩離別情重，噙着泪看父母奔上忙下安顿寝室的背影，哽咽着拥抱道别；又想到今后有佩珩等人日夜作伴，才渐渐不甚伤感。
翌日正式开学。升完旗后一堂英文课下来，宛莹暗想温班程度不过如此，总算安了心。接下来的生物课固然艰深，但老师风趣幽默，深入浅出；虽偶尔开个黄腔，倒也无伤大雅。之后三节便是理化数连堂。宛莹早听父亲说化学李超群老师是中大物化所资优毕业；物理老师叫林长汀，更是辅大物理高材生。待那林长汀上了讲台，宛莹见他俨然一个干瘦版的骆校长；横眉若剑，不怒自威；开口便一通说教：“你们按理是一中的尖子班。但我看过你们之前答的题啊，真的不行的，差很远的！还有人‘牛顿第二定律’不写，偷懒，写什么‘根据牛二定律’，笑死人了！牛二是北宋末年的人，运动三定律不是他发现的！他对物理学的唯一贡献就是跟杨志合作，”说着做个抹脖子的动作，“亲自证明了钢铁强度取决于金属键结构。”宛莹尚未反应过来，许多人已笑倒在桌。林长汀又忙道：“顽笑归顽笑，你们千万不要听我信口说，就在试卷上写：‘林长汀某年某月教的，泼皮牛二是应用物理学家。’不然骆校长又罚我去扫厕所了！”于是全班又笑。随后他并不讲课，只复习国中基础，顺便极尽能事地挖苦直升班前任物理老师；大家毫无良心地哄笑了十余次，直到课后发下一沓齐指厚的材料作业才吓呆了。宛莹此刻终于得知颜槿华所劝不虚，便又烦恼起来。
化学李老师是个年轻些的白面书生，尖声尖气的；却一样不苟言笑，讲到兴起处摇头晃脑、狠敲板书，一绺额发亦随之摆荡；他留的习题虽远比林长汀少，但也有一份两页双面的卷子要求明晚做完。高老师进来上数学课了，宛莹却仍在发愁功课之事；直到听见墙那边咚咚作响，知是李超群又在隔壁砸黑板，才不禁发笑。宛莹之前为考一中，恶补过数学；加上高上治拉东扯西，慢悠悠地讲，因此记得并不吃力。只是转眼到了午间下课铃响，高上治也不急着，拖了好五六分钟堂才准放学。到餐厅去时，大堂早已被十余条人龙挤得水泄不通。
宛莹正在队后踌躇，忽见孟驰云已打好饭菜，人群中看到宛莹便朝她走来。宛莹道：“训导主任不准餐前跑步，你又不听。”驰云笑道：“理他做什么，跑快点他就看不真是谁了。”又回身望了望长队，说，“今天菜好，我打了双份；可惜跑得反胃，妳也别排队了，咱们分着随便吃点吧。”宛莹忧道：“训导主任和纠察队在巡呢。”驰云道：“不相干，咱们并不在这吃。”驰云扣上饭菜，领宛莹便抄小路顺着女舍楼沿走，绕过图书馆，经科学楼的连廊；却不回教室，至对面楼的铁门反锁大间外，推窗翻了进去。宛莹问何处，驰云隔窗笑道：“这是咱们数学竞赛室，里边有冷气。这里现没人用，比课室又更偏，断不会有巡查的。”便去开冷气开门。
驰云揭开餐盘，里面两格蒜炸排骨，两格番茄炖肉，外加在来米饭、白灼时蔬；宛莹嘲道：“就知道吃。”驰云一边分菜，一边笑道：“这鬼地方课业又重，管教又严；不能打电动，不能乱跑，连衣服都不能混穿，终日只剩吃这一个念想了。”
宛莹乘机抱怨道：“你们功课好多！物理化学讲得太快，我都没听懂。”驰云便当空比划不过是这般那般。宛莹叹道：“更晕了。晚自习你慢慢教我。”驰云仍逗她道：“再教妳，高上治捉住又拉我去走廊上去啰嗦。”宛莹急道：“不教我，我就从那走廊跳下去死了！你重色轻友！”驰云笑问：“哪里重色轻友了？”宛莹对着恭班方向摇指笑说：“全级第一你就教得起。”驰云回味一阵，勃然变色，冷冷撂下一句：“我饱了。妳离开时记得锁门，餐盘记得还。”头也不回就走了。宛莹出口后即失悔，却料不到他如此暴怒；此刻错愕无言，也只有草草吃过后，收拾盘子回去恹恹睡了。
宛莹自是无法安眠，早早就回了课室，进门却见一个背硕大书包的面生小姑娘在自己左前方的位子坐下，待宛莹近了又站起来，笑道：“同学妳好。我是恭班的骆小雨。”宛莹早知她是校长女儿，再观她笑时酒窝深攒，虎牙半露，清甜可爱；却含胸羞首，单纯可怜，也特意多问候几句。宛莹落座后便找物理作业看，耳旁只听众人鱼贯进入，小雨起身恭迎不迭。众人知她身份特别，连刘冠廷都比平日拘束。忽听她又小小声说一遍：“你好，我叫骆小雨。”回话却是孟驰云没好气的声音：“妳好，我叫晴天娃娃。”宛莹不禁噗嗤一笑；知他在生自己气，因而又不敢抬头看。驰云经过时也不说话，仅掷下一个纸团。宛莹揉开正面是林长汀的公式页，背面密密麻麻的蓝字全是推导注解。她心一暖，却仍气他态度桀骜，便在其上红笔写了“予唯不受嗟来之食！！！”字样，扔回他座位上去。驰云瞪她一眼，却又无可奈何。
随后国文老师也进来，板书写下姓名“冯仰今”，便住了手。宛莹凭他一张无悲无喜扑克脸，三个四平八稳粉笔字，一时不知好歹。响铃后他依旧不言不语，把名字擦了，又写了十六个词；也不拿花名册，单点孟驰云、莫咏玟、骆小雨、望晓娟上去标音。宛莹看那几组词语连国小生都不会念错，心下便觉这老师水平有限。莫、骆二人都默默注了音，晓娟翻个白眼也瞬间写好了，独孟驰云在那瞄几眼词组，又瞄几眼老师，磨蹭好久才填完下去。
宛莹也在底下暗暗注音；一核对时，却吃惊发现驰云全写错了。冯仰今审视一番，果然瓮声说：“别人都对了。孟驰云站起来。”驰云一脸漠然地起立；冯仰今便批评道：“之前国文课你从不认真上的吗？连‘广场’的‘场’是平是仄都不清楚，还依着在地土话的读音来。看来传言中某些温班国文优秀的人，基础并不一定超过别班同学。”语落,四座一阵偷笑。
再看驰云也不羞，冷笑着诘问道：“传言温班师资全校最佳。冯老师博古仰今，自然远超我们。我不敢班门弄斧，只想问一同注音的别班同学，中华第一部韵书出于几时、叫什么名？”全班顿时雅雀无言；宛莹却知他刁难骆小雨是今日午后不快的馀波，又好气又好笑。驰云逼得骆小雨不答，又见冯仰今仍冷冷观望，并不制止，便接道：“南朝陆法言撰写《切韵》正音，开声韵反切之先河。《切韵》里‘场’，就读澄阳反平声。后世鞑虏南侵，汉音胡化，才有如今所谓的新国音。可我即使按这胡化逻辑推演，‘场’也应读阳平；并不知是谁规定它读上声。”一语未尽，底下在地人鼓掌早已此起彼伏，“一哥”、“天神”之起哄声不绝于耳。宛莹再看同写三人，望晓娟又白眼观天，莫咏玟只摇头浅笑，骆小雨早先已不敢正视，此刻更绯红了脸。
冯仰今依旧不愠不怒，只淡淡道：“这个不在联考范围。现在上课还是依据考纲，你要课外有兴趣，本周刚好有个一年一度的鸣凤杯全市征文大赛；恭班人选已定，温班就让你去发挥啦。”众人齐声叫好。驰云这才知中计，忙辩道：“作文其实是望晓娟的好。”晓娟嫌他出尽风头，此语又不知褒贬，便恨恨道：“他最好，什么当然都是他最好！”冯仰今也一锤定音：“大家都封你为神，就是你啦。”众人深知这等文科竞赛一不保送，二不加分，徒增学校的面子，乐得事不关己；有别人木秀于林，自是最好，于是又来一通黄袍加身的掌声笑闹。驰云何尝不晓众人心态，只能恨骆岳文虚荣喜功，净搞这种无聊事，新仇旧恨次第上涌；此刻却再难推托，心想唯有借此机一心坍骆校长的台，方可泄愤。
隔几日去考场参赛，驰云见恭班选手不出所料真是骆小雨，于是心下复生鄙厌。试卷发下，上注“请以‘西江颂’为题作文，八百字以上，体裁不限，诗歌除外。”驰云素来最嫌恶这类粉饰文章，审完了题，别的不管，只寻思如何捣乱。他看到“诗歌除外”四字，就偏要抬杠写首打油诗。

然而驰云才薄学浅，诗哪里是想些就能写的。他刚开了个头，便续不下去了；挠腮苦想了一个小时，卷子上也还是开头四句：

*西江城外西子桥，桥上何人观浪潮？*
*潮平风起闲花落，落花入水新月摇。*

驰云听到收卷铃响，本欲破罐子破摔，交卷弃考了事。谁知那监考老师早就风闻孟驰云狂傲，看他这几句诗倒有模有样；反正所有卷子明天才统一报送教委，此刻何不作个顺水人情，便悄声教孟驰云继续写，午后再去办公室找她交卷。孟驰云见监考老师如此和蔼，况且自己跟洛为民的过节也跟她无关，这时倒不便寻衅滋事；唯有带着卷子回寝，在午休时分翻着参考书，硬着头皮再凑出了十几联：

*春暄日煦波粼粼，曾照昔时浣纱人*
*对水挽妆沉鱼鲤，扪心颦蹙动舍邻*
*濯至天暗衣衫新，江干归来遇大臣*
*疑是仙葩生天外，修成玉女降凡尘*

*快楫轻舟见吴王，姑苏宫内凤迎凰*
*王君年壮霸一方，？ ？家国飨红装*
*破阵曲终心尚炽，衔觞剑罢刃犹凉*

*晨晨起饰宵宵歌，菱花水月秋几何？*
*？ ？ ？ ？闱墙内，晴数梁燕雨听荷*
*？ ？修河出边隘，*
*一夕响屐迎叩扇，却见归人曳兵戈*

*蛰卧经旬故地起，习流二千向披靡*
*旷夜鏖兵铁甲寒，孤城绝战烽烟紫*
*断米空炊势难衡，散骑西奔如突豕*
*勇霸谁堪异姓侯，偷生叵耐下阶耻*
*短巾覆面孤王死，或言国祸自伊始*
*群情激愤诛红颜，一缕闲愁逐流水*

*鸳鸯未能合冢葬，空余西子清波漾*
*望君来世永清贫，终老莫为王将相*
*只忧泉下销前缘，陈情往事终笑忘*
*徒有后辈悼婉凄，凭栏对月成此唱*

驰云写好歪诗，便立即赶去办公室。交完了卷，又免不了被那老师称颂一番、寒暄几句，如此便误了课点。正匆忙转身回课室，冷不防却被一人堵在路口。驰云一定神──
