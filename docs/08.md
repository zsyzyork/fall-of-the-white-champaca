# 第八回 藏暗室贵公子观影，倚白兰官小姐听琴

话说驰云在回程遇着班主任高秋黎，又听他说宛莹即将转学，备受打击。沮丧之际忽闻简讯铃响，他当是宛莹消息，忐忑打开一看，原来只是车开进西江地界后、市内基地台发送的房产广告；便失望地掷下手机。

孟槐安载了儿子老师一程，不邀饭便是失礼。临近终点，他一连提议了好几处酒楼食府。高秋黎让了一番，仍碍于盛情难却，只好以节后脾胃积滞为由，胡乱找了个路边的便宜鱼庄。下车入座后，高老师又主动揽过点菜的活，只拣了三四样清粥时蔬，便说够了，一意让侍应生下单。孟槐安正担心儿子吃不惯这粗茶淡饭，却见驰云不发一语，连平日不喜的粉葛、西洋菜，也一应胡乱往嘴里送，尽改挑食的毛病；还当是儿子在班主任跟前不敢造次，不禁暗叹：这世上居然有人降伏得住他。茶足饭饱，再客套过几句，孟父便送高秋黎回府，又顺路去医院晃了一圈，方与驰云归家收拾洗漱不提。

再休整一日，便是资优班与毕业班返校补课的日子。时下正值回南天，百物潮润。虽是二月上旬，体表竟有点闷。一中高中部坐落山间，水汽更为丰沛。是日清早，驰云拖拖沓沓地走在校道上。只见天上只有几丝微雨，校道却湿漉可鉴。放眼四望，一片浓白；草色烟光，宛若仙境。宿舍楼群，半截如云；人在高处，雾深不见；只闻上方惨叫声、笑骂声，此起彼伏、不绝于耳。

原来在假期里，一中校方终于响应舆情，准备在各寝室安装空调，以解南国夏热之苦。年前施工队就位，一通流水作业，三下五除二便干完了打洞钻墙、安装支架的杂活。谁知学校采购的冷气机竟迟迟没有发货。包工头等了半天，眼看手下工人一个个返乡去了，唯有待年后劳力充足、设备到位后再作计议。那起工人也是全年仅这一个长假，年关在即，大家都归心似箭，半刻都不愿多呆；老板一放行，他们就像听到放学铃的高中生一样，全脚底抹油、撒腿就跑；竟没一人记得把钻开的洞封回去。天一返潮，山雾便顺着这些孔道汹涌灌入、无微不至。其时，一中校方虽有言明令：放假前需清空宿舍，尤其贵重物品务必带走；却阻不住有人只记得后边半句。那些行李众多、住址遥远、或是没有专车来接的学生，大都嫌弃枕席笨重狼犺，临行只将球鞋耳机随身听一卷，不值钱的傢生一概收入床底；还自作聪明地打着算盘，心想就算工人再穷，也不稀罕这些便宜物件。哪承望放假回来正值换季，那些密封不严的被褥衣物果然多已霉变。诸生回寝一查看，都傻了眼。好在当下不算正式开学，装修又尚待完工；学校只能一边催单，一边暂准大家走读。因此各人有用品污损的，还来得及出外汰换采买。

驰云走上自己寝室，只见里间像是被淹过一般，墙上地下全汪着水。轩宇阿志等众手忙脚乱，受潮物品该洗的洗，该烫的烫；子帆虽未落下行李，却是有洁癖之人，此刻正一边絮絮叨叨地埋怨别人不讲卫生，一边里里外外地拖地拭床。驰云有一卷竹席、一对拖鞋在此。只见席面微微泛绿，也记不得是本来的模样，还是新长的霉苔；那拖鞋虽看不到异常，一摸却也出水；虽说是此前用惯了的东西，但他如今心生硌应，遂性一并处理了，等放了学再去市面上找同款的。幸亏他考完期末就把吉他带回了家，不然照此光景也要被扔掉。其时，各寝室已有人整理完了内务，或凑在一处说笑，或趁着室内窗玻璃上凝着雾滴，便上去用手指写写画画。那刚留的画印，没多久又渐染一层薄雾；以往开关窗留下的指迹手痕，此刻湿了水也依稀显现。从走廊望去，各扇窗上新旧交错，层层叠叠；如同画廊一般，蔚为奇景。

此景虽奇，驰云却无心欣赏；只将垃圾打包好带下楼扔了，便独自一人走去温班。教学楼没份装空调，各间课室在整个寒假里都门窗紧闭，班上倒比寝室里干燥凉爽得多。只是每当有人进入，都要从外带进几股水汽；若是谁进来以后忘记随手捎上门、或是不知死要开窗透气的，课室里都怨声一片。少时，穆秀临进门不关，照例被人呵斥；他却奇思妙想道：“你们反正都在抄寒假作业，还不如将门窗全敞开了，放点雾气进来。这样只要拿出习题册往桌上一摊，没做的地方用水笔乱涂几下。横竖不出一会，字迹就散掉了。等老师收了上去，也看不清哪题认真做过、哪题是在糊弄。”扯淡完毕，又引得大家一通笑骂。

此时，温班同学有的在认真自修，有的在玩笑嬉闹；只有驰云默默地数着来人：三十一名男生悉已到齐，女生来了二十一位，只差宛莹和咏梅未至。眼看就要上课了，只听前座也有人嘁喳喳起来。一人问道：“怎么又不见官咏梅？”另一人道：“怕是之前的病还没好。”起先一人便诧异道：“得的是什么病啊？要请假这样久。怕不是跟她堂哥一样转学省城了吧。”另一人又说：“不可能吧？从未听说她爸爸要升迁。你没看新闻么？今年万众公益行的嘉宾里还有他呢。再说了，还有白宛莹也不在。两人一定是又请了假没错。”前一人竟毫无心肝地羡慕道：“生什么病这么好，下次我也生一个请假去。”旁人打趣道：“那还不如放出消息去，就说你要生个孩子休产假，这样一整年都可以不回来了。”于是又一阵哄笑喧闹，便无人再追究宛莹咏梅去向。其后老师就位，上堂授课，略同既往，不需细述。

中午放学，学校餐厅料定如今人少，又是走读，于是只开了四个窗口，按往常周六日接待少量留宿学生的标准，随缘煮了芥菜鱼糕、冬瓜肉丸两个菜；一副“爱来不来”的派头。诸生固有省事不愿外出的；但大多数还是贪玩，乐得三五成群地上街逍遥。老师一走，望晓鹃便呼朋引伴，要凑四人叫计程车，准备去逛中区新开的商场。子帆随杨父回家吃饭。余人各自心有所属，不久也都散了。只剩驰云在教室倍感凄清，想着若宛莹还在，自己必将带她去别处玩耍的；如今故人已去，不仅出外的兴致阑珊，就连去餐厅的一射之地都嫌太远。

驰云原地看了两句书，发了一会呆，忽然瞥见身侧的窗外闪过一个人。他本不以为意，谁知片刻后又鬼鬼祟祟地飘过两三个身影，怕是有贼；于是不免警觉起来；出去一看，人影虽不见了，湿地上的脚印倒是新鲜。再循迹望向左邻，只见那良班里倒是灯火通明；隔着玻璃水雾，隐约见得投影幕正亮着；又有一人在讲台上，专注用着嵌在教桌里的授课电脑，倒不像是作乱之辈。驰云悄悄拉开后门，探头一瞄；见到台上之人是国中资优班里分流走了的旧同学，便安了心；刚要抽身，却不防被对方抬头盯住，只好打个招呼问道：“你在忙什么，连饭都顾不上不吃了？”台上同学便笑着招手让他进去，说是老师让组队参与科创竞赛、如今正做开题简报；见驰云来了，正有一处想请教他。”驰云闻言，便凑近耳语道：“你趁午休抓紧去玩倒是正经，这么当真做什么？不怕告诉你：这比赛早已内定得奖人选了。只别说是我说的。”那人苦笑道：“我又不读温班了，哪还指望竞赛得奖？不过是被逼交差罢了。不然到时良班比别班少交一份开题报告，班主任丢了脸，又得骂我了。你以为人人都像你一样，有底气顶撞老师的？你若还看得起我们这些人，好歹帮个忙；就当替我省时间了。等会我再跟你讲个好玩的。”

驰云没法，只好去台上操作一番。完事后，那人喜之不禁地谢过他，又诡秘一笑道：“你要是得闲，现在到隔壁艺班去，包你终生不后悔。你只别说是我说的。”驰云问他为何自己不去，那人答说一点钟还要去俭班探女朋友呢。驰云遂独自走出来，见地上泥印果然继续往左通向艺班；且艺班教室无端端地拉紧了窗帘，一看便不对劲。此时正好又有一人神出鬼没，稍稍开了条门缝，泥鳅似地往里一滑，驰云便紧跟着进去。入室迎面只见一团漆黑，唯独投影仪蓝荧荧地亮着。借着这点光，驰云才看清台上操作电脑的，竟是年前躲在厕所里弄鬼的魏梓钧。他刚拔腿要走，魏梓钧却命两个喽啰将前后门都反锁了，又确认了一遍窗帘密合，才放心笑道：“兄弟们，现在马上就要开片了。既然大家让我上来放光碟，我就说一点：咱们看归看，可千万不要喧哗；不然把老师引来了，你我都走不脱。如果有人喊起来，不管出不出事，我都会立即停播，以后谁也别想再让我做这差使了。大家没有问题吧？”驰云见他前番经自己一吓，如今学得谨小慎微了不少，兀自冷哼了一声；却闻四下各处冒出声音，齐说“没有！”又吓了一大跳，竟不知周遭竟黑压压的站满了人。那些回应得响亮的，复被旁人指责道：“刚讲过不要喧哗，你还这么大声做什么？”于是又一阵窃笑。

魏梓钧见自己号令成功、一呼百应，心中相当受用。他身在普通班，此时理论上还有一个礼拜的假；却因身在校管乐队，不免被教练殷洋宁老师要求提早回校彩排，准备在元宵节的万众公益游行里率领一中花车过场。梓钧父亲今日同样需返西江基地复工，只担心这几天儿子在外结交非类、妄生事端，故亦乐见他回学校有人约束。这魏梓钧虽无心排练，倒也宁肯远离父母管教；反正自己在管乐队里收了不少大哥小弟；更难得的是寒假去澳门旅游时，自己趁父母不备，偷偷去黑市上用压岁钱买了几盘最新的正版日本成人光碟，此后便日日当宝贝似的藏在裤兜里，起寝洗漱皆不离身，生恐被发现了挨打。如今重获自由，他想着终于有机会一睹其中女优芳容，便不胜期待；却又觉得独乐乐不如众乐乐，不如借个空教室投映出来，跟小弟们共同学习交流一下，也更显得出自己的胆识作为。只是不知谁走漏了风声，如今连资优班都有人得了消息、寻到这里。他起初还有点生气，后来却一转念道：“能拉资优班的学生下水也不赖。学校平时将他们当宝一样。万一被人捉住了，我就将水搅浑，说光碟是今日在校内拾到、被他们硬逼着放的。学校查不到源头，必不敢重罚他们。有他们挡着，我还更安全些。如果没人来捉更好。就算那样，我也有了他们的把柄。切！我还以为这些‘好同学’多有道德呢，还不是一个两个都猥琐好色？”他一边想着，一边志得意满地插入光碟，按下“播放”。

驰云至此还不知要播的是什么禁片，却单单因为组织者是魏梓钧，便鄙夷得无以复加；可是如今再借故出去，又担心他怀疑自己告密，不得已只好留下一探究竟。他犹自恍着神，影片便已开播。只见片头赫然一块“日本伦理审查协会”的标牌，后边紧跟三大版日文，驰云便猜到了大概。其后一阵电脑特效，光影渐去后，才放出一个姑娘的面部特写。驰云看她眉宇轩昂、姿容英伟，尽管是女子，竟有几分帅气；只是容貌难辨亲疏，总觉有几分面熟，却一时想不起像谁。正寻思着，镜头已拉至全身，原来这是个穿着黑色泳装的模特。不久右下方又转出“小澤マリア”五个大字。驰云涉猎广而不精，之前虽曾试图学过基础日本语回话，但连五十音图都还未记牢就丢下了，如今只有三脚猫的功夫。见到这三个片假名，他只顾自言自语地认道：“小泽──马、里、呜？这名谁起的，这么搞笑。”正疑惑着，身旁一人忍不住扑哧一声道：“你才搞笑呢。人家叫小泽玛丽亚！你连她都不认识。”驰云羞得双颊赤热，幸而黑灯瞎火的不引注目；却还不甘认错，仍试图挑刺道：“无端端起个英文名，也是不伦不类。”那人知识丰富，又普及道：“人家是混血儿。”驰云犹问：“怪不得长这样。是哪里混哪里？”左近一人便低声骂了过来：“大家都守纪律，只听得见你俩在这吵。上面说一句，你说一句。这片子没法放了。非要惹得钧哥生气，你们才开心？”两人吓得忙闭了嘴。

再看片中小泽玛利亚又说了几车子的话，才进展到少儿不宜的情节。众人来了精神，一时赞叹声、咽口水声、拉椅子入座声，极细极微却不绝于耳。驰云观画听音，却不禁愧疚起来，担心宛莹得知了自己今日行状荒诞，以后不知会如何鄙厌看低自己；转而又想道：“她之前不知我看过这种片子，还不是招呼也不打一声就走。往后大家已是两个世界的人了，今生今世能不能再相见还难说；就算哪天偶然遇到，彼此也都生疏了，又有何话可谈呢？”失望至此，他不免自暴自弃；干脆睁大双眼，聚精会神地看起荧幕来。

驰云虽欲放纵，却到底心中郁结。加上这种东洋片子来来回回就那几个场合动作，不仅驰云难以专心，就连在场所有观众看得久了，都渐觉无趣。那魏梓钧在台上也不自在，感觉自己像在唱卡拉OK暖场，却点了首没滋没味的歌，切又不是，不切又不是；再耐心看了几分钟，一晃滑鼠，发现后头还有将近一个半小时的内容，才果断暂停播放、征求台下道：“这样下去时间怕来不及，咱们拣几处紧要的看。”众人也都如蒙大赦，交口答应不迭。魏梓钧随即一通快进，让大家再看了十几分钟方完。

影片放送完毕，魏梓钧把光碟弹出收好、电教设备复位，再拉开窗帘。众人相顾左右，都难堪不已；有立即起身走人的，也有再坐了一会才走的。驰云觉得影片本身倒无可无不可，只是不齿魏梓钧的为人，也嫌自己脸上无光，刚起得来就径直往外走。正要出门，却望见阿志居然在与魏梓钧攀谈。驰云怕阿志也跟这种人学坏了，就编了个事由骗他出来。那魏梓钧尚在里面喊：“哎！以后你们若还想看片，随时来找我！”驰云却想：“网路上资料那么多，谁沦落到非得沾惹上你，那才真是见了鬼了。”便强扯阿志离开，又低声告诫道：“以后你离这种人远一点。”

不知不觉已是下午一点三刻。那些去学校餐厅就食的同学悉已归来，出外的人也陆续返回。晓鹃、若素、轩宇和另一人前后脚进门；想必刚才搭的是同一辆车。晓鹃拎着一个小纸袋，若素空着手，只有轩宇满满当当地提了好些东西。班里众人对别的都不感冒，唯独见了轩宇手上的一大桶炸鸡套餐，一帮没出校的男生便沸腾了，都饿狗般地奔向轩宇讨东西吃。轩宇因父母都出差再外，原本打算中午多买一桶鸡，能管今夜明日四顿饭的；如今见到众人来抢，慌忙把桶举高，制止不迭。却听有人喊道：“我出十块买一根鸡柳！”旁人觉得好玩，便都接连喊价道：“我出十五买一块鸡腿”，“我出两块买一粒鸡米花”，“我出一块买三根薯条！”一只鸡翅甚至能细分成翅尖、翅中、翅根出售；就连蘸料都被预订一空。

轩宇听到有利可图，略微掂量一番，居然真的坐下来，一手收钱、一手交货。最后算了算帐，一个五十块加折扣券花买来的全家桶，拆开零售竟卖了百元有余，把今日车钱饭钱都赚了回来；遂觉难以置信，暗自称奇不已。一抬头子帆又进来了，逢人便问：“你们怎么都在吃炸鸡？也分我一点。”后来得知是从轩宇处买的，旋即面色一沉，丢开不问了。轩宇见状，也不解自己到底哪里值得他生这么大气，至今快两个月了都还没消；正长吁短叹着，阿志又不知从哪里凑到了身前。原来阿志历来不吃早餐，刚才为艺术献身，便连午饭也忘了。此刻闻到炸鸡的油香味，他胃里凭空翻搅，饿得发慌。如今餐厅已收档了，宿舍楼下的超商这几日又不开，他只好问轩宇买一截玉米、或半块面包，总之先撑到放学再说。轩宇哪好意思赚他的差价；随便收了几个钱，就把别人不爱的玉米棒、面包卷、蔬菜沙拉，一股脑地全塞给了他，只叫他不要声张。

阿志自然满心欢喜。谁知他刚一转身，便与正从后门慢慢踱进来的高老师对上了眼。阿志来不及藏，高秋黎便一眼扫到他手中的纸桶。阿志以为他要没收，心想糟糕，不自主地把桶抱紧了些，又赶紧放下了。高老师却只笑说一句：“你中午吃得很好嘛！”便转身去观察别人了。阿志突然记起今日学校是允许外出就餐的，自己干嘛要偷偷摸摸；才心有余悸地捧着桶回座位上去。其余学生被高秋黎一搅兴致，嘴里的炸鸡也都不香了；一个个如同败军被俘似的，垂头丧气地丢下食物，端出书本准备上课，不在话下。傍晚放学众人回家，翌朝再来，一连几日无事。

再说一中校方眼见下周一就全体正式开学，这几日紧催慢赶，终于等到空调送货，忙喊工队复工；机器安装既毕，再让舍监开了半天的除湿，将寝室内气候回复如初，方得始终。这周日本是旧历十五，只是元宵节不算公众假日；兼且西江本地习以冬至为重，宗亲好友平日里也聚多离少，元宵日除了白天的花市和万众行之外，并无特殊节目。如此情形，一中便令全体高中生周日晚回校住宿；只法外开恩免去一节晚修，让外省客家籍的同学从容吃完团圆饭再来。数日来大地回暖，霭消雾散；到周六拂晓，天竟难得地放了晴，全城风和日融，气象台预报周末无雨。市民们有信神的，都说是天公开眼、保庇明日参与公益行的善众不用打伞。一中许多同学也赶在这日回宿舍安顿，以便给元宵夜饭留足时间。驰云亦不例外，前一晚就被父母叮嘱着重拾了行装，自己再顺带把吉他一并背上，以解住校之闷。

放学到家，驰云吃一阵，睡一阵，晚饭后做了功课，再偷空练一回琴，打两回电动，便将一日消磨过去。翌日醒来，他想着在校也是无聊，便顺手把琴也背了回去；此后中午有空便练一段，弹腻了便放进教室后头的储物间里。
