# 第八回 藏暗室贵公子观影，倚白兰官小姐听琴

话说驰云在回程途中鞍马劳神，又巧遇班主任高秋黎，听他说宛莹即将转学，于是备受打击竟不知是睡死还是昏厥过去；抵达西江时才被父亲唤醒。孟父强邀高秋黎吃晚饭，高秋黎难却盛情；为不使他破费，兼因过年时一连几日酒肉炸果下肚，如今脾胃积滞，便提议去了个便宜的渔庄，让店家冲了壶淡茶，煮了一煲鱼片粥，清炒了个地瓜叶，白灼了点莴苣油麦菜；比病号餐还要寡淡。驰云平时哪吃得惯这种没滋没味的东西；只是如今失魂落魄，眼前有什么，便往碗里舀什么。孟父还以为他因为班主任在前，不敢任性挑食；到底还是心疼儿子，又让切了一碟卤水鸭。驰云也仍如行尸走肉一样，机械地夹菜不误。茶足饭饱，再客套了几句，孟父便绕道送高秋黎到府，又顺路去医院晃了一圈，再带驰云回家、收拾洗漱不提。

又休整了一日，驰云便要返校补课。其时正值回南天气，百物潮润；虽是二月中旬，竟有少许闷热。一中高中部在瑶琴山脚，水汽更甚；天虽无雨，校道上却湿湿漉漉。放眼四望，连空气都是一片浓白，只略透着些草木的绿，十步以外就见不到人；整个校园宛若仙境一般。是日清早，驰云拖拖沓沓地走到宿舍区，只听惨叫笑闹声此起彼伏。原来寒假期间，学校在给宿舍在装空调。施工队之前流水作业，在各寝室朝向晾衣间的窗户上打好了洞，装好了外机架子。时近年底，工人跟最后一节课等放饭的学生别无二致，都心不在焉，一到年关便脚底抹油回了乡。过了一周，暖锋北上，山上结的雾气便从窗洞汹涌灌了进来，无微不至。学校虽有言在先，让大家放假前清空宿舍，尤其是务必带走贵重物品；却阻不住有人以身犯险。那些行李太多、住得太远、或是父母种种原因没开私家车来接的学生，大都嫌席被盆桶大件狼犺，临行只把耳机随身听等电子产品一卷，不值钱的傢生都收在床底。学生见识浅，只想着宿舍条件跟工棚也差不多，再穷也不至于偷这些东西；却没料到工队作业马虎，又适逢天气回潮，竟不幸遇着今日这一出。

此周装修队复工，且高一高二只有资优班学生补课，因此学校暂时允许众人走读，等下周装修完了才需留校住宿。只是不少人挂记着寒假前没带走的被褥衣物；一回寝探视，果然发现多数都已生霉，于是叹息痛恨不已。驰云上了自己寝室，只见阿志等众手忙脚乱，受潮之物该洗的洗，该烫的烫；子帆虽未拉下行李，却是有洁癖之人，此刻正一边絮絮叨叨地埋怨别人邋遢，一边里里外外地拖地擦墙。驰云自己只有一卷竹席、一对拖鞋在此；只见席面斑斑驳驳地长了片不知是青霉还是绿苔的物质，黑色的拖鞋虽看不出异样，一摸却也水淋淋的。他觉得恶心，也懒得去擦拭，索性一并丢掉，回头再找家里要新的。

其时，各寝之人有整理完内务的，此时或凑在一处说笑，或见到宿舍的窗玻璃也凝着雾滴，便上去用手指写写画画、尽情发挥。底子好的，就画些古风山水、欧美卡通，甚至有用整个窗扇创作六格漫画的，要么就走笔如飞，写些中英文书法；别的男生也有乱画星星太阳的，简笔蛇龙虎马的，当然也有借机辱骂老师舍监、描摹男女性征器官的，不一而足。新留的画印，没多久又渐染一层薄露；尚未染指之处，也能隐隐看出过往的手迹；新旧交错，层层叠叠。从五楼走廊望去，俨然一面艺术涂鸦长墙，蔚为奇景。

只是此景虽奇，驰云却无心欣赏；打包好垃圾带下楼扔了，便一个人走去温班课室。不久后大家也陆续抵达。在寒假里，各教室关门闭户，没有湿气，倒还干爽阴凉；所以但凡有人进屋不关门，先入课室的都一并叫骂，抱怨外头的雾气被带进来了。驰云一一数着，全班三十一个男生，二十三个女生，只差宛莹未到，便心知高秋黎所言是真，宛莹果然转到别校去了。他固然失落，只是前昨日早已崩溃过了，如今心神日益麻木，倒没觉得比之前难受了多少。

其后老师就位，上堂授课，略同既往；终其一朝，并无可记叙之事。中午放学，学校餐厅因为人少，也只开了四个窗口，按平时周六日接待少许家在乡镇、放假不回的留宿生的标准，只煮些芥菜鱼饼、冬瓜肉丸随便卖卖。因不用住宿，学校也不强求大家在餐厅用餐，只要下午两点准时回来上课就好。而且此日温班第五节又是数学，高秋黎照例拖堂。班上学生固有嫌外出麻烦、舍远求近去餐厅的，也自有不少人乐得上街逍遥。望晓鹃等人呼朋引伴，准备凑够四人，打车去东区新开的商城逛店；教工餐厅不开，子帆遂跟杨父回家吃饭；余人有的钟情汉堡鸡排、有的偏爱拉面寿司、有的想买游戏光碟、有的欲往电子市场，都一哄而散。若宛莹在，驰云必定要带她出校的；如今她走了，驰云便无心玩耍；打算去餐厅，又觉得为今天的饭菜根本不值得走两百多米路；只是留在课室也照样无趣。他正不知该不该走，忽见走廊上有一两人鬼鬼祟祟地经过；等出去看时，人影又都不见了。驰云想，反正闲着也是闲着，不如前去探察一番。他见隔壁良班拉紧了窗帘，遂知其中有鬼，于是悄悄开后门溜了进去。

驰云进了良班，只见讲台上拉下了荧幕，开着投影；有两人站在授课电脑边上，一人敲着键盘，一人用着手柄，正对战着科乐美世界足球竞赛，玩得不亦乐乎。下面坐着四五个男生，有良班的，也有别班的。大家也都看得入胜、时不时加油叫好、拍桌叹息。台上两人发现了驰云，却都听闻他平时也桀骜叛逆，绝不可能给老师打小报告；就算打了，老师也会把他当作首号嫌疑犯，又不敢责罚他，自然更不会殃及从犯了──因此也不甚理会。驰云见众人大笑大闹，倒关切起来；见有一人是国中资优班的同学，便凑过去耳语道：“你们也太嚣张了，就不怕骆校长偷偷摸摸寻过来？”那人却不以为意，指着窗边一人道：“不紧要的。你看他不就在盯梢么？要是老师来了，咱们就一个切屏──那边开了个空的投影简报──就说正在设计班刊呢。”驰云又指台上的人道：“横竖也留点神。万一老师看到手柄，岂不就穿帮了？”那人也不直接回应，反而神秘一笑道：“就算老师要抓，也轮不到我们。告诉你个事，还有好多人正在隔壁礼班看成人电影呢！”

两人正说着话，台上两人的比赛便已分了输赢。胜者友好地邀驰云上去对打；也有拉他下水、有难同当之意。岂知温班诸多男生都不爱做假期作业，周末寒暑假无事一身轻，不是在做竞赛题，就在疯狂打电子竞技，以资调剂；虽然别无长物，一手电动倒是玩得炉火纯青；这些人中又以驰云为甚。驰云上去，一两分钟就灌了那人三个球，又怕别人笑话他，后面自己只能让着打。那对手也看出驰云在让球，自是扫兴；草草比完这盘，便撂开手柄让别人上。旁人不想跟驰云玩，但又怕冷落他；一时你谦我让，尴尬万状。驰云见势，唯有主动下了台，说自己只想观战；于台下再看了两盘，觉得还是无趣；趁着有人进球、大家不注意，就顺原路溜了出去。这时正好撞见班上一人偷偷摸摸地进了礼班课室，自己便也尾随而入。

此刻礼班与良班一样，也是窗帘紧闭、黑灯瞎火的。只是投影机虽已打开，幕布上却仅有视窗系统的桌面墙纸。再定睛一看，电脑边上站着的人居然是陆启钧。陆启钧见又有两人进入，也不急着放送；而是先让人锁了前后门，再压低声音、宣布纪律道：“兄弟们，我再说一遍：大家既然是来这里看片的，那咱们就好好看片。这是日本的正版AV，如果看到激动的场景，千万不要喧哗；如果跟你们之前看的谜片有什么不同，也不要大惊小怪。若是招来老师，咱们谁都跑不掉。所以只要有人喊起来了，我就立即停播，以后都再也不放了。大家听明白了吗？”台下众人便齐说‘明白’，声音或洪或细；那些嗓音嘹亮的，又被旁人悄声指责“都讲了不要喧哗，你还这么大声做什么”，于是大家又窃笑起来。驰云再看台下并无人入座，桌椅后边的空地倒是齐刷刷站了两行人，其中半数都是温班的。504寝室里，他倒不见其余室友，唯独欧阳志在列。阿志瞥见驰云，便对他咧嘴会心一笑。

陆启钧见自己号令成功、一呼百应，心中相当受用。他虽不在温良恭俭四班，却因科创任务，如今要提前归校，也算进了半个资优班了；值此机会，他准备与四个班的学生建立关系，多发展几个人脉，却又不知要怎样融入──若论古典文学、先锋画作，自己一窍不通；若谈明星八卦、游戏动漫，又怕被人看轻；至于足球政治、星座穿搭等话题，各自也都受众寥寥；思来想去，只有一种艺术，不分民族阶级，据调查全世界有七成男人爱看，其余三成都是大话精。如此谋划着，他寒假去澳门旅游时，便趁父亲玩二十一点、母亲排队买榴莲雪糕之际，以上厕所为由，偷偷去黑市上买了一盘最新的日本成人片回来；只是又怕双亲发现、自己挨打，于是把包装拆了，碟片藏进裤袋里，洗澡睡觉都不离身，直至今日返校才拿出来，一为普济众生，二为扬名立万──黑社会杀人放火的事，自己当然不敢去碰；但如今冒着被开除的风险，在校公然组织播放色情内容，足见自己亦是有胆做大事的人；至于那些赏脸观影的，即便不能跟随自己出生入死，至少也是一路人，稍施小恩小惠，便可收而用之。这样想着，他又清点了一遍台下人数，发现资优班竟有两三成的男生在这，管乐队里的温班前现队友也都来捧场，甚是自得；这才把碟片放进光驱，开始放送电影。

驰云见组织者是陆启钧，便鄙夷得无以复加；只是这时若是出去，又担心被怀疑告密。况且驰云没看过正版成人片；宛莹刚刚出走，他又有了自暴自弃之意，便索性留下来一探究竟。影片伊始，镜头只给到某处外景，只见一幢两层小楼，楼外树影摇曳而已；却已有一众男生早已看得伸头张口、眼神发直。下一个场景，就是一个衣衫齐整的女子坐在沙发上接受摄影师的采访。驰云没学过日文，听不懂她在说什么，便去问身边人这艳星是谁。那人看了驰云一眼，难以置信地笑道：“小泽玛丽亚啊。这你都不认识？”

原来世上的某些肤浅男生，包括驰云在内，都有个不良风气──那就是非要千方百计地证明自己脑袋有多聪明，同时最看不起那等只会啃教科书、死记硬背、笨鸟先飞之人。为与这等人划清界限，他们非要在歪门邪道上彰显自己，以自证不是书呆子。别人副科认真听讲，他们偏要睡觉；别人晚修做作业，他们偏看杂志小说；别人思想单纯，他们就用淫邪包装自己。明明背地里学得比谁都刻苦，却总爱装出一副云淡风清的模样，好将成绩归功于自己的智商。之前驰云歧视骆小雨，也正因此故。驰云并未出过国，周围又没有一干不争气的宵小，哪里弄得到真正的东洋动作片；稍能记住几个德艺双馨的日本女星，也是拜网上流传已久的一首歪诗所赐──

*苍穹对井犹自空，星落天川遥映瞳*  
*小溪流泉映花彩，松江孤岛一叶枫*  

即使驰云久仰苍井空、松岛枫大名，他却没看过她们的任何一部作品──从这点而言，驰云和他所鄙视的那类只景仰曹雪芹狄更斯、却不读红楼梦双城记的俗人，其实也半斤八两──那小泽玛丽亚还是刚出道的新人，名字又不跟前诗押韵，他则更是闻所未闻，只感叹原来日本艺人也流行取英文名。驰云再看她眉宇轩昂、姿容英伟；说是长相独特，又观之可亲；说是秀色可人，又总觉哪里不对，便再问刚才身边那人道：“她怎么长这样啊？”那人道：“听说是个混血儿。”驰云又问：“所以是哪里混哪里？”那人便不耐烦道：“啧。我怎么知道？你问七问八的，到底是看还是不看？上面音量本来就小，你这一说话，大家都听不清了。”

驰云便不敢作声，却想着，国内老师授课时，绝无此等殊荣；只有这种时候的日本老师，才能让男生们不但自动全神贯注，还自觉互相督促。再一抬头，只见玛丽亚老师仿佛越聊越热，便开始动用一切手段防止中暑。当下，周遭全是咽唾沫、吞口水的声音；陆启钧也不站在电脑旁了，而是寻了张椅子坐下细看。后排站着的人，见状也纷纷入座。等即将看到紧要处时，电影导演却娴熟地运用起模糊叙事的手法，留下一场悬疑，以供回味。有的观众便失望地叹起了气，心想这片子还不如自己的私藏精彩。再后来便是经典的动作搏斗场面；其精微之处无法言表，故而略去。

约摸坐了一堂课的功夫，影片才放送完毕。众人鼓掌道过谢后，有立即走人的，也有回味了一会再走的。驰云对电影本身倒无可无不可，只是不齿陆启钧的为人，大有不食周粟之意；临走时便不言谢，径直起身正准备出门，却见阿志居然在与陆启钧攀谈。驰云见子帆已中了他的圈套，怕阿志也跟这种人走近学坏了，就编了个事由要他跟自己一块走。那陆启钧尚在笼络阿志说：“哎！以后想看片子了，再来找我！”驰云却想：“网上素材那么多，谁若今后沦落到非要看你的录影不可，那才真是见了鬼了”，便强拉阿志离开，悄悄告诫道：”以后这种人你离远一点。“

不知不觉已时至下午一点半。去学校餐厅就食的同学都已归来，那些出外的也在陆续返回。不一会，晓鹃、轩宇、若素、还有若素的一个室友同时抵达；想必是刚才一起搭的便车。晓鹃拎着两个手提袋；轩宇一手提着精品店的纸袋，另一手捧着个炸鸡全家桶。那帮没出校的男生闻到油香，便沸腾了，都争先恐后地找轩宇讨东西吃。轩宇父母晚上要出门应酬，他原本打算中午多买一桶鸡，今夜明早不至于饿肚子的；见到众人来抢，慌忙把桶举高，制止不迭。此时又有人道：“我出十块买一对鸡柳！”旁人有的从众，有的觉得好玩，便都接连喊价道：“我出三十买一组鸡腿！”，“我出一块买一粒鸡米花！”，“我出一块买三根薯条！”一只鸡翅甚至能细分成翅尖、翅中、翅根出售；就连蘸料都被预订一空。轩宇一听有利可图，居然真的坐下来，一手收钱、一手交货。等别人银货两讫，阿志也凑了过来，说想买一截玉米。轩宇却不好意思赚他差价；随便收了他几个钱，就把别人没买的玉米棒、面包卷、蔬菜沙拉，全一股脑地留给他，只叫他不要声张；最后点了点帐，一个五十块加折扣券花买来的全家桶，拆开零售竟卖了将近百元，把今日车钱晚饭钱都赚回来了；只觉难以置信，暗自称奇不已。
