# 第七回 欲解小衅杨父和事，因听片语高徒失魂

话说嘉宇听闻科创竞赛之事，却错过了毛遂自荐的良机，正失悔着，忽又灵光乍现，便悄声对驰云道：“不瞒你说，其实我一直有个创业点子，用在这个竞赛上，再好不过了──如今咱们国家那么多人炒股。可是六成散户都在亏损，三成不赔不赚，只有不到一成赚钱的。为什么？因为散户消息不通，还不理性，不爱分析，全凭感觉。乱买的股票小赚一笔，就以为自己是股神了，然后越涨越买；等涨到头了，他又不肯抛售，非要亏本了才割肉，越跌越卖；所以屡战屡败，被庄家耍得团团转。其实，资本会用脚投票，主力资金的动向全藏在技术指标里；而历史又在不断重演，只要比对以往的交易和经济数据，就大概率预测出下一次大涨是什么时候。”

驰云向来不信这些邪说，只冷笑道：“电视上的分析师就天天干这个。要真有你说的容易，他们早该盆满钵满了，哪还需要西装革履、东访西窜，上节目耍嘴皮子挣钱？”

嘉宇耐心解释道：“你听我说，那些分析师自己也是人。只要是人，就有七情六欲，就不可能完全理性。有时候你甚至不知道，他们荐的股是真的科学分析过；还是随便乱找几支，再千方百计地生成有利数据来圆谎？你想，一支股票一天怎么也有上百个节点、上千股成交，上市以来数据至少上万。如果只靠蛮力，一个分析师一年能别说搞懂一支股票，就算能把所有数据归档做成图表，就已经不错了，哪来的精力给你每周荐股？所以我觉得，那些所谓的股评员，十有其九都徒有虚名；你看他们履历就知道了──在正经券商做过核心业务的就没几个；有人甚至连电脑都不会用，看走势全靠肉眼。真正的高手，像是顺大数院、辅仁经院的学生，一毕业就有华尔街公司开百万年薪抢人，哪有功夫……”

驰云打断道：“说了这么多，你的意思无非是人脑迟钝、人心叵测，只有电脑最好了。丢一大堆数据给电脑分析，你坐在旁边喝咖啡。咖啡喝完了，结果就量化出来了，就能预测涨跌了。是不是这样？”

嘉宇没听出讥讽，反而激动道：“我就知道，这话跟你说准没错！别人要么没学过建模，要么没写过程式，要么没读过金融；解释得再详细，都是对牛弹琴；只有你有奥数基础，参加过资讯竞赛，文科功课又好。别人说到科技创新，顶多只能想到物理化学发明，什么足球机器人、什么奈米材料石墨烯；咱们这个项目可是跨学科的，空前绝后。如果做成了，且不说获奖保送；单是把成果商业化，就是好大一笔钱──你想，南省好几百万股民，只要一两成人用咱们的成果，每人再交一两块钱的年费，咱们的收入就赶超华尔街精英了。”

同窗三年，嘉宇只知驰云杂学旁收、涉猎广泛；却不知他博而不精、一无所长。驰云听了这番愿景，倒是自知才能有限；只是为了掩饰心怯，反而嘴硬道：“我听得懂，只是觉得不可行。单说最首要的，你打算上哪下载股票资料去？若券商肯分享交易数据，那还好办；万一不肯，你是花大钱去买呢？还是去找骇客撞密钥、写爬虫？”

嘉宇乐观道：“万事开头难。只要攻克这关，之后就全是咱们的强项了。”驰云却再泼冷水道：“开头难，之后更难──咱们线代微积分才刚入门，你哪来的信心能找对数学模型？动不动就上万个数据点、万亿次的复杂演算，一台家用电脑要算多久？还有抽象的总体经济指标，你又要如何量化？”嘉宇踌躇满志道：“这些都可以学。说不定用功个一年半载，就都学会了呢？”驰云只推辞道：“你若有余力，你只管学。我又有奥数，又有功课，再钻研这些有的没的，不用吃饭睡觉了。之前我退管乐队，正是因为没时间；如今谁也别想拉我下水了──你想建模型，就去找穆秀临或子帆；想写程式，就去找阿志他们；总之别找上我。”

嘉宇原本想着，子帆昨夜态度坚决，大有绝交之势；自己必须争取驰云、阿志等室友，才进可促成和解，退可有地自容。他将秘而不宣的创业想法坦而告之，也有笼络驰云之意。谁知驰云神态委顿、兴致阑珊；意气心神，迥异往日。嘉宇擅于察言观色，跟一众女生又亲近；起初虽用心打探科创息，忽略了驰云的反应，可一番对谈下来，眼见驰云心不在焉，他便也猜着了七八分，于是大度宽慰驰云道：“也罢也罢。我听晓鹃、素素她们说了，昨晚宛莹在哭，你又在发脾气，也不知道是小事大事，是吵架还是闹分手。都说‘攘外必先安内’，现在你肯定无心干别的。这比赛才刚开始，不急这一刻。你先把内事处理好，科创的事以后再说。”

驰云听了这话，先是疑惑，后又一惊，随即矢口否认道：“越说倒越科幻了。我跟宛莹又不是男女朋友，分的哪门子的手？”嘉宇却道：“这就怪了。大家都知道你和宛莹过去青梅竹马，又看见如今又出双入对；无论男女生，都觉得你们正在交往啊……”一语未了，只听见底下有小碎步上楼的声音。两人只道是高老师回来了，都吓得噤了声。楼下来人却并非高秋黎，而是赶来上第一堂课的李迅永。李老师见二人鬼鬼祟祟地靠在墙角，便责备道：“大家都在早读，就你俩在外边聊天是吧？还不快进去上课。”嘉宇还分辩道说高秋黎让他们出来的，李老师却义正词严道：“高老师明确指使你们旷我的课吗？那我回去倒要问问他，数学班主任不让学生上中文课，是哪门子道理。”二人一向拗不过李迅永，更怕他告黑状，唯有从权行事，回去上课。

驰云心中本来只牵挂着宛莹，此刻却纷乱繁杂了起来。他和宛莹自幼熟络，又久别重逢，话题比别人更多，原属正常；之前相处得轻松自在，也是因为二人赤诚无邪，只将彼此当作挚交，并未多心。如今经嘉宇一说，他却自省起来，不知自己对宛莹的好感是止乎友谊，还是参杂了别的情愫。驰云之前虽懵懂有过情感经历，但也仅限与诗言的一段，所以只知拿前任来比──那会他俩还是幼稚国中生，刚进叛逆期；两人是初恋，又面临竞赛考试、分班升学诸多压力；因此精神紧绷、不知轻重；情路也大起大伏、惊险轰烈。自己如今与宛莹则无非日常笑闹而已，见面毫不觉得忐忑悸动、心荡神驰；如果这也叫做恋爱，未免太寡淡了。再说诗言敢爱敢恨，爱了就主动告白，伤了就决绝分手；整段感情全由她主导，自己全程都在招架。宛莹和自己都不似这般炽烈耿直，如若真要表白，甚至不知如何开口；做惯了无话不谈的密友，更不知以后要做怎样的眷侣，才能益彰亲近、又不失自然。此外，当初跟诗言分手，两方都深受打击；诗言强忍情伤，被激得失血晕倒，闹得满校皆知、谣谮四起；自己也因此被父亲屈打住院，至今心结未解。即使宛莹如今同意交往，也难保将来不会撒手，到时多年的情谊被挥霍抛掷，着实可惜；双方父母又是故交，万一自己辜负了宛莹，或她有个三长两短，怕是连上代人都要反目成仇。如此想来，两人倒是继续做朋友最为保险。

只是驰云自打听了嘉宇的话，便有了心理暗示。他明明准备就此打住，再不去想宛莹，却禁不住满脑海里全是她；明明心中自证了一百遍，自己和宛莹只能是普通朋友，却又疑神疑鬼，不明白自己为何纠结此事、念念不忘。课室里宛莹不在、梅珩不知所踪；若回去找嘉宇理论、反而坐实了他的论断；至于别人，按嘉宇所说，都已认自己已经恋爱，自己平时又并不和他们深交，倘因此事突然造访，倒像自己多爱张扬显摆似的。温班虽众，驰云竟无一人可以倾诉，课间也只能独自烦恼惆怅，神思一直恍惚，上午五堂课全没听进去。中午放学，他浑浑噩噩地归了寝，也还止不住去想那没边没际之想、忧那无根无由之忧。其时，子帆未回，嘉宇一直在上铺看手机；阿志亲历了这两人的口角，生怕自己被骂多嘴，此刻仍然大气都不敢喘；余人见宿舍自昨晚就死气沉沉的，也不知是谁和谁闹了矛盾，因也早早躺上床铺，咸缄其口。未几午休铃响，四下微鼾渐起。只有驰云思绪翻覆、躯肢辗转，一整个中午都没合过眼。

等到下午竞赛集训，驰云的瞌睡才姗姗来迟。课前时分，他晃出宿舍、途经图书馆、爬上二楼，正准备右转进入奥数教室，却见组里众人还在往上走；抓过一人打听，那人竟道：“咱们主场搬了！”驰云便问：“好端端的，怎么就要换地方上课？”那人老实回话道：“传说学校要把二楼的课室改装成什么创新实验室，奥数培训改到五楼了。”驰云听了这错愕消息，才稍稍回到现实。只听他火冒三丈道：“左也创新，右也创新。一个黑心比赛，居然有脸踩咱们的场。”那人也只好诺诺应和着。

行至顶楼，果然有间大课室敞门以待，奥数组诸生鱼贯而入。驰云打量此处并不算残破，只是毫无空调设备；可容纳百人的空间里，只有寥寥两盏吊扇──便对旁人嘀咕道：“地理老师说过，高中部本来就夏热冬凉。是谁这么缺德，那么多好课室不拣，非拣这个鬼地方上课。如今挨冻，多穿几件也算罢了；只是到夏天又要热死。”旁人都笑道：“哪考虑得了这么远。”驰云见无人未雨绸缪，便郁郁不乐；只一心等高秋黎出现，自己要据理陈情。谁知高老师久久未至，后来竟是杨子帆同他父亲杨祖润老师寻了进来。

原来高秋黎早前接到兄弟急电，说老父病危、恳请速返。自古忠孝两难全，他顾不得骆校长脸色，赶紧请了个事假，就坐长途车回乡去了。杨父此番代班，正是受他所托。这杨老师之前也是个闻名遐迩的奥数教练。只是骆校长讲求物尽其用；对如此杰出人才，教学之余还不忘摊上巡修监考等琐务。杨父历年殚精竭虑，几年下来体力透支；前年体检，心脑肝肾、血压尿酸各指数更是全面飘红，不得已才将奥数教练一职让贤给了高秋黎，自己牛刀杀鸡，改教高一和文科班去了。

因子帆之故，杨父早已听闻穆秀临、孟驰云、佟嘉宇等人大名；今日出山，自然也心痒，打算好好试他们一试。别人都还好，只有驰云状态全无，却又不敢打瞌睡，只能一直强撑着，好不辛苦。嘉宇知道驰云心病，自己又好表现，少不得替他挡了许多。

因高秋黎不在，驰云也忘了陈情之事；与子帆虽有龃龉，但毕竟杨父在场，又不便算账。倒是那子帆，不知今日是浪子回头还是人逢喜事，趁嘉宇套瓷杨父、聊得兴起之际，竟主动找上驰云，又是赔笑，又是作揖；说待会见到嫂子，还要负荆请罪。驰云在长辈眼皮底下，也不好意思穷追不饶；又见他摧眉折腰、态度诚恳，自己唯有和解了事。

捱到下午放学到家，驰云连鞋也顾不得脱，便径直奔向房间，找出手机，拨了宛莹号码，怎知却无人应答；再打开电脑，登上聊天应用，也不见她在线。这天夜里，他再打了几轮电话，可还是一次都接不通；给她传的简讯也没有回复。如是一来，他心里便又多了几分不切实际的猜想。本来一桩小事，只因他闲极生愁、庸人自扰，以至整个元旦假期都没睡好。直到节后返校傍晚，在校道偶遇着宛莹平安归来，驰云才松了口气。

宛莹歇了三日，伤已好了九成，心口也不大疼了。只是她也不知，同驰云闹成这般，何时才是了局。她见驰云也望向自己，却进退失措、欲言又止，一副委屈巴巴的模样，可气可怜；便主动设个台阶，上前轻声问道：“嗳，你有什么要说的吗？”驰云闲着三天，却只一味胡思乱想，竟未深思过重逢宛莹时该说什么。眼下他先道了歉，顺带为子帆求了情；弯来绕去，唯独不敢吐露心迹。他原想说，自己这几日一直挂记着她，一刻不停，就快要得神经病了──但这话太像烂俗电视剧的肉麻台词，只好作罢。打了一会腹稿，他又打算说，两人从小玩到大，一直要好，又何至于此；若因诗言子帆等外人疏远了，不但可惜，也让人笑话，话至嘴边，却也嫌煽情，不像是自己口吻，反倒像极了嘉宇会说的，便又硬生生吞了回去。驰云思来想去，终于只讪笑道：“话说，今后可不可以不要拿诗言奚落我了？你同她根本不熟，无谓成日一直提她，弄得大家尴尬，倒不和气了。”

宛莹听了这句，却解读成“你我虽比别人更早认识。可你国中读了外校，又拉下了功课；同我不仅早已生疏，简直就是两路人了。如今你非但完全不了解诗言，也不了解我。我与诗言无论好或不好，也至少有过刻骨铭心的专属回忆；此刻还轮不到你一个外人插嘴”，因又想道：“你终于承认，诗言是你心口永远的朱砂痣。那我就给你承诺：从今过后不再开你俩的玩笑。只是但凡你心中还存在哪怕一丝友谊，也不会看我都这样了，还偏要提起她来。可见你心里装着诗言，就容不下其他人了。枉我傻傻地把你看做个知己，原来只是一厢情愿。”如此想着，便又灰了九分心，失落道：“好，我都听着了。你还有什么别的话要说么？”

驰云真以为诗言之案经已了结；自己其它心迹又不便讲，犹豫了一会便说：“除这件以外也没了。”宛莹只道他对自己早已无话可诉，难免心内阵阵酸楚、眼里簌簌掉泪，强忍着啜泣，哽咽道：“算我白认识了你。”说话间便碎步跑开。

驰云向来拙于人情世故，此刻当然不解宛莹心事。他跑上前去将她拦住，转眼却又质问道：“你这阵子又是怎么了？我到底哪处惹到了你，明说出来岂不好？一番两次的，要哄到几时才算完？”宛莹哭道：“原来这叫哄我！既是这样，不如不要说话，还能少操些心。”驰云急得失控，赌气道：“那你到底要我怎么办！是不是要我不说不笑、不烦你了，你才开心？那样的话，大家也唯有敬而远之了。”宛莹听了此语，便止了哭，在原地怔了良久，才慢慢抬头，凄怆道：“好，那就……”她尚未说足半句，却又哽住了，一手掩了口，一手抹着泪，转身绕过驰云，继续前奔。

宛莹一路疾走，驰云尾随申辩；两人追逐至温班廊下，不巧正遇着杨祖润、杨子帆父子。原来元旦节前，科创教练叶堪双召走子帆后，又把杨父拉过来游说。杨父一介明星教练，怎会不想儿子继承衣钵。只是他早已耳闻奥数组人才辈出；节前自己也亲自验过，穆秀临、佟嘉宇等人底子果然变态。若在别地还好，只是南省数学联赛竞争白热化，统共三十个一等奖名额，竟有四五万人逐鹿；若论获奖难度，甚至不亚于考四大名校。子帆如果孤注一掷押注联赛、以求加分保送，实在太过冒险。另一方面，这科创竞赛也是新生事物，他还看不出各种门道。虽有叶堪双在旁画饼，但到底该信几分，他却毫无成算；只怕儿子退了奥数、误了学业，到头来保送应大却是空头支票。元旦国定假日，学生不准补课，老师倒有干不完的活。杨父本就有恭班作业要改、年终总结要写、期末卷子要出；这会代班，温班又要备课、奥数又要集训，便彻底没了自由。放假三日，他竟在办公室电脑前坐了三日；深夜戴月归眠时，子帆还在外边K歌宵夜；清晨披星上班前，子帆却又睡得正香。直到如今返校，杨父才趁着晚修前校园电视放新闻的空档，去温班寻儿子商议，于是正巧碰上驰云两人。

杨父见他俩一个眼红气促、泪痕满面，一个脸色铁青、含恨眉间；便存心要调和调和。他与宛莹不熟，却一心想跟驰云套近乎，便上前拍肩憨笑道：“孟驰云啊，怎么竟然对女同学发起脾气来啦？”子帆见父亲越俎代庖，此刻尴尬万分，又生怕牵出自己撞人之失，遂埋怨道：“爸！人家两个的事，你又乱凑个什么热闹。”这杨老师也是个心直口快、天真烂漫的人。他回过神来，一拍脑袋，便试图开个玩笑缓解尴尬，说道：“喔！有情况啊，哈哈！你看我这眼神。以前总听人说，孟驰云找了个我们班的──嘿嘿，我还专门在恭班附近偷偷盯过一阵子，就是好奇想看看：唉，孟驰云的对象会是谁？结果呢，明里私下一次都没见他来过。我说奇怪了，这个人也太绝情了吧？今天才总算明白……”一语未了，子帆忙把他推到楼梯间，催道：“你快回你的恭班值晚自习去！给你代几堂课还不过瘾，居然管起我们来了。骆岳文发的年终奖金值几个钱，你操这份闲心做什么。”宛莹听了杨父之语，方知驰云并未藕断丝连；虽暗自惭愧，却又不知当年诗言之事到底闹得多大、以至人尽皆知。她见自己在众人跟前颜面尽失、无地自容，遂再受刺激，于是夺门而入，伏在桌上偷潸。

这边驰云也一肚子的气，不明白为何全世界都找自己麻烦。眼看宛莹归座嘤泣，他生怕再惹众人口舌，只好悄悄从后门兜进课室。原本两人闹不愉快，总预着官梅珩会来化解；然而今次吵得毫无退路，又一直未见梅珩归来，更不知这僵局如何能破。子帆虽有意补过，不懈安抚，奔忙两端；孰料谁都不肯卖帐。他浪费了许多唇舌，中途又再被杨父与叶教练叫走，终究无力劝和。

驰云宛莹都还少不更事。驰云耿介，自不用说；那宛莹虽不似诗言气盛，却也是个自爱要强的。两人刚受折挫，如今就都怕再遭伤害。此间又没有第三个人介入转寰，他俩虽心中难安，却不知要怎样和解。这边驰云想着，宛莹是女孩子、又初来乍到的，自己理应多让着她；却又担心自己一句话讲错，又惹得她无名火起、不可理喻地对自己撒气。那边宛莹也宁愿自己委屈些，也多一事不如少一事，只是如今就快赶上月里那几天；前番受的气着的凉，这会都应在身上，痛得死去活来，难免情绪不佳；那驰云又永远一副高傲骄矜的架子，着实气人。此时期末考试将近，她理科仍大面积欠账；不同驰云和好，就更无人帮她补习──梅珩请假，晓鹃性急，周围大多女生同她一样自顾不暇，若找别的男生讲解，又怕驰云多心。林韫之的功课倒好，挨得又近；可宛莹请教过她几次后，她晚修便只温习文科。听林韫之的室友说，她回寝后倒夜夜挑灯，物理昨夜写到凌晨一两点；她既已如此艰辛，自己更不该打扰。如此，驰云宛莹一味地赌气矜持，都期待对方先软化，却又不断失望；两人一直僵持到期末考完、寒假回家，都竟未再说一句话。

展眼除夕将近，驰云父母身为主治医师，难得竟同时获批年假，自然开车举家返乡、探候老人。医生假期固短，国高中生倒有教育部规定的三周寒假；然而西江教育署求好心切，为抓升学率，连国定假期也敢讨价还价、打个七折；一中再提前半周补课，竞赛组又抽两日集训，如此算下来驰云正月初五便要返校，倒正好与父母的上班日期相重。驰云在长身体，平日住宿就睡不够；这一路又舟车劳顿加拜年应酬；祗好每有闲暇就抓紧补觉，一个寒假竟白白睡了过去。

眼见又要开学，驰云便怏怏不乐，返程时空望着窗外，一路无话。等到驶过雪岭关，展眼春色夹道、梨蕊满野，其华似霰、其皎若莹，他心内才又泛上宛莹的名字，祗不知她此刻如何。驰云想着两人冷却了大半个月，再见时也理应冰释前嫌了；这才打起半分精神，倚着车窗，斟酌着重逢时的每句说话。待到出了果林，近了市肆，孟怀安把车开进服务区，寻了个排档就餐，他仍边吃边望着远山发呆。

驰云正嚼着饭，冷不防身后一个雄浑男声道：“哟！这不是孟医师么！”孟驰云拧头见是高秋黎，惊得差点跌出座位，心想他素日潜伏跟踪的功力再强，也无理由明察暗访到自己在这。听父母与他攀谈几句，才知他料理好家事后，不迟不早，偏也拣了这日乘长途客运返回西江。这会趁着中转休整，他赶紧下车抽烟解瘾，谁想就恰好就撞见驰云一家。驰云父母见到儿子班主任，哪有不客气奉承的；又要约吃饭，又邀他坐顺风车同走。高秋黎因盛情难却，竟真也应承下来；从大巴车取出了行李，便随驰云一家摆布。驰云讨厌他身上的残烟味，人情礼节也不顾，祗缩到后座角落里远远躲着。

高秋黎偏不放过驰云；与孟父神侃了半刻钟，便转身道：“孟驰云啊，听说我不在的这几天，温班状况很多咧。”驰云深知老师们故作此语，若非捉到了把柄，就是准备套口供。他以为高秋黎言指自己与宛莹，一时心悬嗓口，却装糊涂道：“老师讲的是官梅珩生病的事？”孟母也插话说：“是官议长的女儿罢？我们同事接诊过她几次，都说这孩子家教好，一点公主病都没，完全不像这种人家出身。”

高秋黎点头道：“这个官梅珩的确难得。虽然天资不是最出众，可是非常扎实低调，病得那么重才去请假，住了院还不忘复习功课。可另外有些人就不一样啦，”驰云以为他言指自己，便低头佯装系鞋带；再听他继续说道，“我刚回老家不久，有天早上五点多骆校长就打电话骂过来，说穆秀临他们寝室深更半夜聚众打牌，聪明都不用在正经事上，净搞些歪门邪道。这件事我回去还要仔细追究──孟驰云啊，这学期高老师就不会再讲情面了咧！”驰云父母忙问自己儿子有无劣迹，又劳烦他严格管束；高秋黎却笑道：“孟驰云他们寝室倒好，不仅多才多艺，还很团结，”说着又重提一遍欧阳志夜哭之事。驰云未料到这番谈话也能跌宕起伏、峰回路转，至此才敢换个坐姿、长松口气。
