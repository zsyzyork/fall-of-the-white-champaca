# 第七回 欲解小衅杨父和事，因听片语高徒失魂

话说叶堪双找人参与科创竞赛，嘉宇因错过了毛遂自荐的良机，正失悔着，忽又灵光乍现，便悄声对驰云道：“不瞒你说，其实我一直有个创业点子，用在这个竞赛上，再好不过了。”驰云让他有话快说，他便笑道：“如今咱们国家，二十个人里就有一个在炒股买基金。以后的发展趋势，也肯定是跟美国看齐，大家退休金都存在股市，全民都是股东。可是散户炒股，六成都在亏损，三成不赔不赚，只有不到一成赚钱的。为什么？因为散户消息不通，还不理性，不爱分析，全凭感觉。乱买的股票小赚一笔，就以为自己是股神了，然后全仓做多，越涨越买；等风向一转，他又心存侥幸，不肯抛售，非要等亏本了才割肉，越跌越卖；这样循环往复，被庄家耍得团团转。其实，即使没有内部消息，只要会看财报和技术图形，比较以往的收支、K线、成交量，就能判断出大致走势；尽管不能保证稳赚不赔，但至少比意气用事可靠多了。只是绝大多数人都太懒，放着正路不走，非走捷径，所以才亏得惨。”

驰云向来不信这些邪说，便冷笑道：“电视上的分析师就天天干这个。要真有你说的容易，他们早该盆满钵满了，哪还需要西装革履、东访西窜，上节目耍嘴皮子挣钱？”

嘉宇耐心道：“你听我说，那些分析师自己也是人。只要是人，就有七情六欲，就不可能完全理性。有时候你甚至不知道，他们荐的股是真的科学分析过；还是随便乱找几支，再千方百计地生成有利数据来圆谎？你想，一支股票一天怎么也有上百个节点、上千股成交，上市以来数据至少上万。如果只凭人力，一个分析师一年能别说搞懂一支股票，就算能把所有数据归档做成图表，就已经不错了，哪来的精力给你每周荐股？我觉得，那些所谓的股评员、分析师，十有其九都徒有虚名；你看履历就知道了──要么学校一般，要么专业一般，也没几个在券商做过核心业务的；有人甚至连电脑都不会用，对比K线全靠肉眼。真正的高手，像是顺大数院、辅仁经院的学生，一毕业就有华尔街公司开百万年薪抢人，哪有功夫……”

驰云打断道：“说了这么多，你的意思无非是人脑迟钝、人心叵测，只有电脑最好了。丢一大堆数据给电脑分析，你坐在旁边喝咖啡。咖啡喝完了，结果就量化出来了，就能预测涨跌了。是不是这样？”

嘉宇没听出讥讽，如遇知音地激动点头道：“我就知道，这话跟你说准没错，也只有对你能说！奥数组最受器重的也就秀临和你，你之前又搞过资讯竞赛，文科功课又好。找遍全班，别人要么没学过建模，要么没写过程式，要么没读过金融。解释得再详细，都是对牛弹琴。你想，大家要么偏理，要么偏文，没法把知识融会贯通；就算参赛，也毫无新意，最多弄个新材料新器械，什么奈米技术、机器人，要么就研究自闭小孩的早教，多数人根本用不上。咱们这个项目可是普惠万千大众的。如果做成了，获奖保送那还另说；单是把成果商业化，把分析模型移植到各种股票应用程式里，那咱们也发了。你想，南省好几百万股民，只要一两成的人用咱们的模型，每人交一两块钱的年费，那收入也赶上华尔街精英了。”

同窗三年，嘉宇竟还未看清驰云底细，不知道他虽杂学旁收、涉猎广博，却多而不精、毫无造诣。驰云听了这番愿景，深知自己才能有限；为掩饰心怯，反而嘴硬道：“我听得懂，只是觉得不可行。单说最首要的，你准备上哪下载股票资料去？若券商肯分享交易数据库，那还好办；万一不肯，你是去买资料呢？还是去找骇客撞密钥、写爬虫？”

嘉宇乐观道：“万事开头难。只要攻克这关，之后就全是咱们的强项了。”驰云却再泼冷水道：“开头难，之后更难──咱们刚上高一，线代微积分才刚入门，你就有信心能用对用好数学模型？总体经济的抽象指标，你要如何量化？动不动就上万个数据点、万亿次的复杂演算，一台家用电脑要算多久？若是拟合结果毫无意义，你该怎么办？”嘉宇道：“这些都可以学。说不定用功学个一年半载，就都会了呢？”驰云只推辞道：“你若有余力，你只管学。我又有奥数，又有功课，再钻研这些有的没的，不用吃饭睡觉了。当初我退出管乐队正是因为这个；如今谁还想拉我下水，可不能了──你想建模型，就去找穆秀临或子帆；想写程式，就去找阿志他们；总之别找上我。”

嘉宇原本想着，子帆昨夜态度坚决，大有绝交之势；自己必须争取驰云、阿志等室友，才进可促成和解，退可有地自容。他刚把秘而不宣的创业想法坦而告之，也有笼络驰云之意。但驰云却神态委顿、兴致阑珊；意气心神，迥异往日。嘉宇毕竟擅于察言观色，跟一众女生又亲近；起初虽一心打探科创消息，没空注意，但一番对谈下来，驰云的心事他也猜着了七八分。此刻要务是收买人心，他随即大度笑着宽慰道：“也罢。这比赛才刚开始，并不急在这一刻。我听晓鹃、素素她们说，昨晚宛莹哭了，你又在发脾气。我也不知道是小事大事，是吵架还是闹分手。都说‘攘外必先安内’，现在你肯定无心干别的。如果要人调停，我去帮忙说和；如果你想宣泄，元旦假咱们散心喝酒去。”

驰云听了这话，先是不以为意，只想着自己被人编排取笑了一整年，也不差他一个；再看嘉宇一脸认真，又愣了一愣，随即冷笑否认道：“我跟宛莹又不是男女朋友，分的哪门子的手？你们今日造谣我跟诗言宛莹，明日又跟梅珩小雨。我再混帐，也不至于这么朝三暮四，见异思迁吧？”

嘉宇却不解道：“这就怪了。诗言的事早已翻篇，咱班同学都知这是个话题炸弹；你自己不说，别人有谁敢提？我也没听说过梅珩、骆小雨跟任何人的八卦。只有你和宛莹，过去青梅竹马，如今又这么亲密；所有人都看得出来你俩关系不一般啊。”

驰云还分辩道：“你们死读书，成天只会‘青梅竹马’、‘青梅竹马’地乱用成语。李白的原诗‘郎骑竹马来，绕床弄青梅’，下句‘两小无疑猜’怎么翻译？明明两个人思想纯洁、不用避嫌，哪里就关系不一般了？”

驰云本想掉个书袋、镇住嘉宇的；正背着诗，脑中却自动联想到了再下句的‘十四为君妇’；这才自悔引喻失义，语落便脸红起来。嘉宇却只重重拍了驰云的背两下，意味深长道：“行。那你好自为之吧……”一语未了，只听见底下有皮鞋上楼的声音。两人以为是高老师回来了，都吓得噤了声。

谁知来人并非高秋黎，而是赶来上第一堂课的李迅永。李老师见二人鬼鬼祟祟地站在墙角，便责备道：“大家都在早读，怎么就你俩还在外边说话？还不快进去。”嘉宇正解释是高秋黎让他们出来的，李老师却打断道：“高老师明确指使你们旷我的课对吧？那我回去倒要问问他，数学班主任不让学生上中文课，是个什么道理。”二人一向拗不过李迅永，更怕他告黑状，唯有从权行事，回去上课。

驰云心中本就牵挂宛莹，如今听了嘉宇之言，便更加纷乱繁杂。他原本只是单纯觉得，与宛莹相处起来轻松自在；如今却也怀疑起来，不知自己对她的好感到底是单纯无邪的，还是超越了友谊。回想起关于自己跟诗言、小雨、梅珩等人的碎语，貌似除了宛莹，也确实没有别人提起过；又记得宛莹曾说，喜欢一个人才会针对他，也不知她三番两次打趣自己，是说明她也有意，还是自己的错觉。如若两人真要交往，自己也不知道要怎么表白；做惯了无话不谈的好友，更不知表白之后要如何相处，方能益彰亲近、又不失自然。

驰云如此神思恍惚，上午五堂课全没听进去。课室里宛莹不在、梅珩不知所踪，若回去找嘉宇、又坐实了他的说法；驰云竟无一人可倾诉，课间也只能独自胡思乱想。午后放学，他浑噩吃过饭归寝，也止不住去想那没边没际之想、忧那无根无由之忧。一众室友里，子帆未回，嘉宇一直在上铺看手机；佟、杨口角时阿志在场，此刻他生怕自己被骂多嘴，自然气都不敢喘；余人见宿舍自昨晚就死气沉沉的，也不知是谁和谁闹了矛盾，因也早早躺上床铺，皆不敢出声。未几，午休铃响，四下微鼾渐起。只有驰云思绪翻覆、躯肢辗转，一整个中午都没合过眼。

当日下午照例是竞赛集训。驰云睡意姗来，懵懵地晃出宿舍、穿过图书馆，刚上二楼，正准备进奥数教室，却见组里众人还在往上走。抓过一人来问，那人却道：“咱们主场换了！”驰云问道：“好端端的，怎么就要换地方？”那人道：“我也不知。听说之前的课室要改做科创办公室，奥数课室搬去五楼了。”驰云火冒三丈道：“左也创新，右也创新。一个黑心比赛，居然有脸踩咱们的场。”那人也只好诺诺应和着。

行至顶楼，果然有间大课室敞着门，穆秀临、佟嘉宇等人鱼贯而入。驰云打量此处并不算残破，只是毫无空调设备；一百人的场地，却只有寥寥两盏吊扇；于是同旁人嘀咕道：“地理老师说过，高中部本来就夏热冬凉。是谁这么缺德，那么多好课室不拣，非拣这个鬼地方上课。如今挨冻，多穿几件也算罢了；只是到夏天又要热死。”旁人都笑道：“哪考虑得了这么远。”驰云见无人未雨绸缪，便郁郁不乐；只一心等高秋黎出现，他要独自陈情。谁知高老师久久未至，倒是杨子帆同他父亲杨祖润寻了进来。

原来高秋黎今朝接到兄弟急电，说老父病危、恳请速返。自古忠孝两难全，他又是长子；于是也顾不得骆校长脸色，紧急请了个事假回乡。杨父此来，正是受他所托，暂带几轮奥数集训。这杨祖润老师之前也是个闻名遐迩的教练。只是骆校长讲求物尽其用；对如此杰出人才，教学之余还不忘摊上巡晚修、监月考之琐务。杨父历年呕心沥血、任劳任怨，几年下来体力早已透支过度；前年体检，他心脑肝肾、血压尿酸各指标全面飘红，这才将奥数教练一职让贤给了高秋黎、自己教高一和文科班去了。因子帆之故，杨祖润早已久闻穆秀临、孟驰云、佟嘉宇等人之名；今日上课，自然也心痒，打算好好试他们一试。可惜此刻驰云全无状态，又没好意思打瞌睡，只能强撑着，好不辛苦。只有嘉宇知他心病，又力图表现，少不得替他挡了许多。因杨父在场，课间休息时驰云也不好意思找子帆算账。倒是那子帆不知是幡然悔悟、还是人逢喜事，趁嘉宇套瓷杨祖润、聊得兴起之际，竟主动找上驰云，又是作揖，又是道歉，又说等见到宛莹，还要再负荆请罪。驰云见他摧眉折腰、态度诚恳，长辈跟前也不好意思穷追不饶、小事化大，唯有选择和解。

等驰云熬到放学回家，连鞋也顾不上脱，便径直奔向电话。他拨了宛莹手机号码，怎知却无人应答；打开电脑，登了通讯应用，也不见她在线。之后驰云再打了几轮，却依然接不通；传的简讯也杳无回复。如是一来，他心里便又多了几分不切实际的猜想。本来一桩小事，只因他闲极生愁、庸人自扰，把嘉宇之言翻来覆去地品味了上千遍，心里暗示便不知强化了几分；以至整个元旦假都没睡好。直到节后返校傍晚，在校道遇着宛莹平安归来，他才松了口气。

宛莹歇了三天，伤已好了九成，心口也不大疼了；只是她也不知同驰云闹成这般，何时才是了局。宛莹见驰云也看着自己，却进退失措、欲言又止，一副又可气又可怜的模样，便主动设个台阶，上前轻声问道：“嗳，你有什么要说的吗？”驰云先道过歉，顺带为子帆求了情；弯来绕去，唯独不敢吐露心迹。他原想说，两人从小一直玩到大，比别人都要好，若是因为诗言子帆等外人而疏远了，不但可惜，也让人看着笑话。话至嘴边，却又嫌煽情，像极佟嘉宇的口吻，便硬生生吞了回去；又想说多日未见，甚是思念，自己寝食不安，全为了她，却更肉麻了；终于只讪笑道：“话说，今后可不可以不要拿诗言奚落我了？你同她根本不熟，无谓成日一直提她，弄得大家尴尬，倒不和气了。”

宛莹听了这句，却解读成“你我曾比别人更熟。可你国中读了外校，同我早已生疏。你不仅根本不了解诗言，也不了解我。我与诗言无论好或不好，至少已有过刻骨铭心的专属回忆；此刻还轮不到你一个外人来插嘴”，因又想道：“你终于承认，诗言是你心口永远的朱砂痣。那我就给你承诺：从今过后再不拿你与她开玩笑了。只是但凡你心中还存在哪怕一丝友谊，也不会看我都这样了，还偏要提起她来。可见你心里装着诗言，就容不下其他人了。枉我傻傻地把你看做个知己，原来是一厢情愿。”如此想着，便又灰了九分心，失落道：“好，我都听着了。你还有什么别的话要说么？”驰云真以为诗言之案经已了结；自己其它心迹又不便讲，犹豫了一会便说：“除这件以外也没了。”宛莹只道他对自己早已无话可诉，难免心内阵阵酸楚、眼里簌簌掉泪，强忍着啜泣，哽咽道：“算我白认识了你。”说话间便碎步跑开。

驰云向来拙于人情世故，此刻当然不解宛莹心事；忙跑上前去将她拦住，却又质问道：“这阵子又是怎么了？我到底哪处惹到了你，明说出来岂不好？一番两次的，要哄到几时才算完？”宛莹哭道：“原来这叫哄我！既是这样，不如不要说话，还能少操些心。”驰云急得失控，赌气道：“那你到底要我怎么办！是不是要我不说不笑、不烦你了，你才开心？那样的话，我也唯有敬而远之了。”宛莹听了此语，便止了哭，在原地怔住了；良久才慢慢抬头，凄怆道：“好，那就……”尚未说足半句，却又哽住了，一手掩了口，一手抹着泪，转身绕过驰云，继续前奔。

宛莹在前疾走，驰云在后尾随；两人追逐至温班廊下，不巧遇着正在说话的子帆与杨祖润。原来元旦节前，科创教练叶堪双召走子帆后，又把杨父拉过来游说。杨父一介明星教练，怎会不想儿子继承自己衣钵；但他早闻奥数组人才辈出，节前自己亲自验过，穆秀临、佟嘉宇等人果然变态。若在别地还好，南省数学联赛却近乎白热化，统共三十个一等奖名额，竟有四五万人在争，获奖难度不亚于考四大名校；如要孤注一掷押注联赛、以求加分保送，实在太过冒险。只是这科创竞赛也是新生事物，他还看不出各种门道。虽有叶堪双在旁画饼，但到底该信几分，他却毫无成算；只怕儿子退了奥数、误了学业，到头来却没法兑现保送应大的空头支票。元旦假里，学生不准补课，老师倒一堆的活。杨父本就有恭班作业要改、年终总结要写、期末卷子要出，这会再摊上了温班备课、奥数集训，便彻底没了时间；放假三日，他竟在办公室电脑前坐了三日；直到节后带晚修，才趁着校园电视放映一周新闻的空档，跑去寻儿子商议；正巧就碰上驰云宛莹两人。

杨父见他俩一个眼红气促、泪痕满面，一个脸色铁青、含恨眉间；高老师不在，自己便存心要调和调和。他与宛莹不熟，却一心想跟驰云套近乎，便上前拍肩憨笑道：“孟驰云啊，怎么竟然对女同学发起脾气来啦？”子帆见父亲越俎代庖，此刻尴尬万分，又生怕牵出自己撞人之失，遂埋怨道：“爸！人家两个的事，你又乱凑个什么热闹。”这杨老师也是个心直口快、天真烂漫的人。他回过神来，一拍脑袋，便又开起玩笑道：“喔！有情况啊，哈哈！你看我这，以前总听说孟驰云找了个我们班的──嘿嘿，我还专门在恭班附近偷偷盯过一阵子，想看看：唉，孟驰云的对象是谁？高材生眼光怎么样？结果呢，明里私下却一次都没见他来过。那时我还奇怪，今天总算明白了──喂！眼光不错嘛……”一语未了，子帆忙把他推到楼梯间，催道：“你快回你的恭班值晚自习去。代了课还不过瘾，现在居然管起我们来了。骆岳文又不给你发代课奖金，你操这份闲心做什么。”宛莹听了杨父之语，才知驰云并未藕断丝连，便有几分惭愧；却又不知当年诗言之事到底闹得多大、以至满校皆知，又深受刺激；只觉无地自容、颜面尽失，早已夺门而入，伏在桌上偷潸。

这边驰云也一肚子的气，不明白为何全世界都找自己麻烦。眼看宛莹归座嘤泣，他生怕再惹众人口舌，只好悄悄地从后门兜进课室。原本两人闹不愉快，总预着官梅珩会来化解；然而今次吵得毫无退路，又一直未见梅珩归来，更不知这僵局如何能破。子帆虽有意补过，此刻奔忙两端、不懈安抚；孰料谁都不肯卖帐。他浪费了许多唇舌，中途又再被杨父与叶教练叫走，终究无力劝和。

驰云宛莹毕竟少不更事，又都是气盛要强的。两人各碰过一次壁，就都害怕再受打击。此间又没有第三人介入转寰，他俩虽心中难安，却不知该怎样和解。这边驰云想着，宛莹是女孩子、又初来乍到的，自己理应让着她；只是担心一句话讲错，又惹得她无名火起，折挫他的自尊。那边宛莹也愿自己委屈些，只想多一事不如少一事，只是如今就快赶上月里那几天；前番受的气着的凉，这会都应在身上，痛得死去活来，难免会有坏情绪；那驰云又永远一副矜持高傲的架子，着实气人。此时期末考试将近，她理科仍大面积欠账；不同驰云和好，就更无人帮她补习──梅珩请假，晓鹃性急，周围大多女生同她一样自顾不暇，若去找男生讲解，又怕驰云多心。林韫之的功课倒好，挨得又近；可宛莹请教过她几次后，她晚修便只温习文科；听她室友说，她回寝后倒夜夜挑灯，物理昨夜写到凌晨一两点。听说了她如此辛苦，宛莹更不敢打扰。

---

两人各有隐衷，以至直到寒假放假回家，谁也都没有理过谁。展眼除夕将近，驰云父母难得获批同时串休。当医生的最知争分夺秒；有此旧历年假，自然开车举家返乡、探候老人。医生假期固短，国高中生倒有教育部规定的三周寒假；然而西江教育署求好心切，为抓升学率，连国定假期也敢讨价还价、打个七折；一中再提前半周补课，竞赛组又抽两日集训，如此算下来驰云正月初五便要返校，倒正好与父母的上班日期相重。驰云在长身体，平日住宿就睡不够；这一路又舟车劳顿加拜年应酬；祗好每有闲暇就抓紧补觉，一个寒假竟白白睡了过去。

眼见又要开学，驰云便怏怏不乐，返程时空望着窗外，一路无话。等到驶过雪岭关，展眼春色夹道、梨蕊满野，其华似霰、其皎若莹，他心内才又泛上宛莹的名字，祗不知她此刻如何。驰云想着两人冷却了大半个月，再见时也理应冰释前嫌了；这才打起半分精神，倚着车窗，斟酌着重逢时的每句说话。待到出了果林，近了市肆，孟怀安把车开进服务区，寻了个排档就餐，他仍边吃边望着远山发呆。

驰云正嚼着饭，冷不防身后一个雄浑男声道：“哟！这不是孟医师么！”孟驰云拧头见是高秋黎，惊得差点跌出座位，心想他素日潜伏跟踪的功力再强，也无理由明察暗访到自己在这。听父母与他攀谈几句，才知他料理好家事后，不迟不早，偏也拣了这日乘长途客运返回西江。这会趁着中转休整，他赶紧下车抽烟解瘾，谁想就恰好就撞见驰云一家。驰云父母见到儿子班主任，哪有不客气奉承的；又要约吃饭，又邀他坐顺风车同走。高秋黎因盛情难却，竟真也应承下来；从大巴车取出了行李，便随驰云一家摆布。驰云讨厌他身上的残烟味，人情礼节也不顾，祗缩到后座角落里远远躲着。

高秋黎偏不放过驰云；与孟父神侃了半刻钟，便转身道：“孟驰云啊，听说我不在的这几天，温班状况很多咧。”驰云深知老师们故作此语，若非捉到了把柄，就是准备套口供。他以为高秋黎言指自己与宛莹，一时心悬嗓口，却装糊涂道：“老师讲的是官梅珩生病的事？”孟母也插话说：“是官议长的女儿罢？我们同事接诊过她几次，都说这孩子家教好，一点公主病都没，完全不像这种人家出身。”

高秋黎点头道：“这个官梅珩的确难得。虽然天资不是最出众，可是非常扎实低调，病得那么重才去请假，住了院还不忘复习功课。可另外有些人就不一样啦，”驰云以为他言指自己，便低头佯装系鞋带；再听他继续说道，“我刚回老家不久，有天早上五点多骆校长就打电话骂过来，说穆秀临他们寝室深更半夜聚众打牌，聪明都不用在正经事上，净搞些歪门邪道。这件事我回去还要仔细追究──孟驰云啊，这学期高老师就不会再讲情面了咧！”驰云父母忙问自己儿子有无劣迹，又劳烦他严格管束；高秋黎却笑道：“孟驰云他们寝室倒好，不仅多才多艺，还很团结，”说着又重提一遍欧阳志夜哭之事。驰云未料到这番谈话也能跌宕起伏、峰回路转，至此才敢换个坐姿、长松口气。

驰云父母既为搏感情，也防打瞌睡；言笑不绝于路，与高秋黎相谈甚欢。不觉间就过了碧潭、总算在晚高峰前入了省城
