# 第七回 欲解小衅杨父和事，因听片语高徒失魂

话说叶堪双找人参与科创竞赛，嘉宇因错过了毛遂自荐的时机，正失悔着，忽又灵光乍现，眼见四下无人，便悄声对驰云道：“不瞒你说，其实我也一直有个点子，用在这个科技竞赛上，再好不过了。”驰云让他有话快说，嘉宇便笑道：“如今咱们国家，二十个人里就有一个在炒股买基金；西江经济发达，股民只会更多。以后的发展趋势，也肯定是跟书里说的美国一样，大家连退休金都放在股市里，全民都是股东。可是，咱们散户炒股，六成都在亏损，还有三成不赔不赚，只有不到一成赚钱的。为什么会这样？因为散户消息不通，又不理性，不作分析，全凭感觉。随便买的股票偶尔小赚一笔，就以为自己是股神了，于是全仓做多，越涨越买；等市场风向一转，又心存侥幸，不肯抛售，非要行情见底了，他才割肉；这样反反复复，被庄家耍得团团转。其实，历史不断在重复之前的错误。即使没有内部消息，只要看得懂财报和技术图形，比较以往的收支、K线、成交量，也大致能判断出股票什么时候该涨，什么时候要跌；尽管不能保证稳赚不赔，但至少比意气用事可靠多了。”

驰云向来不信这些邪说，便冷笑道：“电视上的分析师天天就在干这个。照你说的这么简单，他们十有其九都早该盆满钵满了，哪还需要西装革履地东访西窜，上节目靠嘴皮子挣钱？”

嘉宇解释道：“你听我说，那些分析师自己也是人。只要是人，就有七情六欲，就不可能完全理性。有时候你甚至不知道，他们荐的股真的客观筛选过；还是先乱点鸳鸯谱，再去各处找有利的数据来圆谎？你想，一支普通股票一日就有上百个节点、上千手的成交；自上市以来，少说都有上万的数据。毕竟，如果只凭人力，一个分析师一年能别说搞懂一支股票，就算能把所有数据归档做成图表，就已经不错了，根本没精力给你每周荐股。所以我觉得，电视上那些所谓的股评家，十个有十一个徒有虚名。你看他们的履历就知道了，大多要么是一般学校，要么是一般专业，也没几个在大券商做过核心业务；有的人甚至连电脑都不会用，比对图线全靠肉眼──与其听他们的，我还不如自己弄呢。真正的行家，像是顺大数院、辅仁经院毕业的精英，一毕业，华尔街马上就开百万年薪抢着要了，哪有功夫……”

驰云打断道：“说了这么多，你的意思无非是人脑迟钝、人心叵测，只有电脑最好了。丢一大堆数据给电脑分析，你坐在旁边喝咖啡，咖啡喝完了，结果也就量化出来了，就能拿去预测涨跌了，是不是这样？”

嘉宇没听出讥讽之意，喜形于色，激动道：“正是！我就知道，这话跟你说准没错，也只有跟你能说！在奥数组里，大家看重的就是秀麟和你两个，你之前又搞过资讯竞赛，文科功课又好，课外知识也多。找遍全班，都没有像你这样的。别人要么没学过数学建模，要么没写过电脑程式，要么没听过金融术语。我解释得再详细，都是对牛弹琴。你想，绝大多数同学要么偏理，要么偏文，没法把知识融会贯通；就算参赛，也毫无新意，最多弄个新材料新器械，什么奈米技术、石墨烯；要么就是教小机器人踢踢足球，普通人根本用不上；咱们这个项目可是直接造福万千大众的，立意又高，又开先河。如果做成了，获奖保送那还另说；单是把成果商业化，把分析模型移植到各种证券交易的应用程式里，那咱们也发了。你想，南省好几百万股民，只要一两成的人用咱们的模型，咱们再收每人一两块钱的年费，那收入也赶上华尔街的顺天、辅仁高材生了。”

嘉宇与驰云同窗三年，却尚未看清他的底细，不知这个孟驰云虽杂学旁收，却多而不精；固然各科俱有涉猎，可论其成就造诣，实连皮毛都算不上。驰云听了这番愿景，深知其中奥义，非自己所能执掌；为掩饰心怯，却嘴硬道：“我听得懂，只是毫无兴趣。单说最首要的，你准备上哪下载股票资料去？若券商肯分享交易数据库，那还好办；万一不肯，你是去买资料呢？还是去找骇客撞密钥、写爬虫？”

嘉宇乐观道：“万事开头难。只要攻克了这关，之后就全是咱们的强项了。”驰云却再泼冷水，道：“开头难，之后更难──咱们刚上高一，线代微积分才刚刚入门，你就有信心能用对数学模型？总体经济的抽象指标，你要如何量化？若是拟合结果毫无意义，你该怎么办？动不动就上万个数据点、万亿次的复杂演算，一台家用电脑要算多久？”嘉宇道：“这些都可以学。士别三日，当刮目相看。说不定再过一年半载，就都会了。即便一无所成，也学到了高等数学，并不算亏。”驰云只推辞道：“你若有余力，你只管学。我又有奥数，又有功课，再钻研这些有的没的，不用吃饭睡觉了。当初我退出管乐队，正是因为这个；如今谁还想拉我下水，可再也不能了──你想建模型，就去找穆秀麟或子帆；想写程式，就去找阿志他们；总之别找上我。”

嘉宇原本想着，子帆昨夜态度坚决，大有绝交之势；自己必须争取驰云、阿志等室友，才进可促成和解，退可有地自容。他刚把秘而不宣的创业想法坦而告之，也有笼络驰云之意。只是驰云一直神态委顿、兴致阑珊；意气心神，迥异往日。嘉宇毕竟擅于察言观色，跟一众女生也亲近；虽起初听到科创竞赛消息，头脑一热没注意，但一番对谈下来，驰云心事自己也能猜着七八分。此刻他一意收买人心，便大度笑着宽慰道：“也罢。这比赛才刚开始，并不急在这一刻。我听晓鹃、素素她们说，昨晚宛莹哭了，你又在发脾气。我也不知道是小事大事，是吵架还是闹分手。正所谓‘攘外必先安内’，你先把这事处理好了，别的再慢慢商议。如果要人调停，我去帮着说和说和；如果你想宣泄，明天兄弟几个陪你散心喝酒。”

驰云听了这话，先是不以为意，只想着自己被人编派取笑，一年以来，竟未停过，如今嘉宇也来凑这个热闹；看他一脸认真，不似说笑，又怔了一怔，随即冷笑一声，矢口否认道：“越说倒越科幻了。我跟宛莹又不是男女朋友，分的哪门子的手？你们今日造谣我跟诗言，明日又跟梅珩宛莹。我再混帐，也不至于这么朝三暮四，见异思迁。幸好我不是女生，不然面子就别想要了。”

嘉宇听言，却困惑道：“这我就不懂了。诗言的事早翻篇了，咱班同学都知这是个话题炸弹；你自己不说，别人谁敢提起？我也没听说过梅珩、骆小雨的跟任何人的八卦。只有你和宛莹，过去青梅竹马，如今又这么亲密。大家都有眼睛，怎么可能看不出来你俩是一对？”

驰云还分辩道：“你们死读书，成天只会‘青梅竹马’、‘青梅竹马’地乱用成语。李白的原诗‘郎骑竹马来，绕床弄青梅’，下句‘两小无疑猜’怎么翻译？明明是目的单纯、用不着避嫌疑的两个小孩，哪里就成恋人了？”

驰云本想掉个书袋、镇住嘉宇的；正背着诗，脑中却自动联想到了再下句的‘十四为君妇’；这才自悔引喻失义，语落便脸红起来。也不知嘉宇是没读过这诗，还是单纯不想抬杠。此刻他只拍了拍驰云的肩，自我圆场道：“行吧。你说是什么，那就是什么。用高老师上数学课的术语──‘这个恋人咧，是一一映射。两小无猜就不一定；两者只是有交集，并不是子集来的嘛！’人又不活在旁人嘴里。你们在不在这个交集以内，别人说的都不算，只有你俩心知……”一语未了，只听见底下有人上楼。两人都以为说曹操曹操到，是高老师回来了，吓得赶紧噤了声。

谁知来人并非高秋黎，而是语文李迅永老师赶来上第一节课。李老师见二人鬼鬼祟祟，便责备道：“大家都在早读，怎么就你俩站在外边说话？还不快进去。”嘉宇正解释是高秋黎让他们出来的，李老师却打断道：“高老师明确指使你们旷我的课对吧？那我回去倒要问问他看，数学班主任不让学生上中文课，是个什么道理。”二人一向拗不过李迅永，更怕他告状，唯有见机行事，归位上课；倘或之后挨高秋黎批评，也好归咎于他。

驰云心中本就牵挂宛莹，如今听了嘉宇之言，便更加纷乱繁杂。他原本只是单纯觉得，与宛莹相处起来轻松自在；如今却也怀疑起来，不知自己对她的好感到底是单纯无邪的，还是超越了友谊。回想起关于自己跟诗言、小雨、梅珩等人的碎语，貌似除了宛莹，也确实没有别人提起过；又记得宛莹曾说，喜欢一个人才会针对他，也不知她三番两次打趣自己，是说明她也有意，还是自己的错觉。如若两人真要交往，自己也不知道要怎么表白；做惯了无话不谈的好友，更不知表白之后要如何相处，方能益彰亲近、又不失自然。

驰云如此神思恍惚，上午五堂课全没听进去。课室里宛莹不在、梅珩不知所踪，若回去找嘉宇、又印证了他说法可能没错；驰云竟无一人可倾诉，课间也只能独自胡思乱想。午后放学，他浑噩吃过饭回寝，也止不住去想那没边没际之想、忧那无根无由之忧。一众室友里，子帆不在，嘉宇一直在上铺看手机；佟、杨口角时阿志在场，此刻他生怕自己被骂多嘴，自然气都不敢喘；余人见宿舍自昨晚就死气沉沉的，也不知是谁和谁闹了矛盾，因也早早躺上床铺，皆不敢出声。未几，午休铃响，四下微鼾渐起。只有驰云思绪翻覆、躯肢辗转，一整个中午都没合过眼。

当日下午照例是竞赛集训。驰云睡意姗来，懵懵地晃出宿舍、穿过图书馆，刚上二楼，正准备进奥数教室，却见组里众人还在往上走。抓过一人来问，那人却道：“咱们主场换了！”驰云问道：“好端端的，怎么就要换地方？”那人道：“我也不知。听说之前的课室要改做科创办公室，奥数课室搬去五楼了。”驰云火冒三丈道：“左也创新，右也创新。一个黑心比赛，居然有脸踩咱们的场。”那人也只好诺诺应和着。

行至顶楼，果然有间大课室敞着门，穆秀麟、佟嘉宇等人鱼贯而入。驰云打量此处并不算残破，只是毫无空调设备；一百人的场地，却只有寥寥两盏吊扇；于是同旁人嘀咕道：“地理老师说过，高中部本来就夏热冬凉。是谁这么缺德，那么多好课室不拣，非拣这个鬼地方上课。如今多穿几件也算了，到夏天又要热死。”旁人无此远虑，都不理会。驰云见没人懂得未雨绸缪，便郁郁不乐；只一心等高秋黎出现，他要独自陈情。谁知高老师久久未至，倒是杨子帆同他父亲杨祖润寻了进来。

原来高秋黎今朝接到兄弟急电，说老父病危、恳请速返。他难辞长子义务，于是也顾不得骆校长脸色，请了个紧急事假回乡。杨父此来，正是受他所托，暂带几轮数竞集训。这杨老师之前也是个名教练，一度声噪西江。祗是骆校长讲求物尽其用；对如此杰出人才，教学之余还不忘摊派上巡晚修、监月考之琐务。杨父历年呕心沥血、任劳任怨，几年下来自然体力透支过度，前年体检心血肝肾各指标全面飘红，才将数竞教练一职让贤给高秋黎、自己退居文科班。杨父因子帆之故，早已熟识孟驰云、佟嘉宇两人；上课自然也更为关照。可惜今日驰云全无状态，又没好意思打瞌睡，撑得好不辛苦。佟嘉宇知他心病，少不得替他挡了许多。

驰云熬到放学回家，连鞋也顾不上脱，便径直奔向座机拨了宛莹号码。怎知她手机偏在这时无人应答，通讯软体也不在线上；想是病得太重、一时住院了也未定。再晚些打过去依旧不通，驰云心里便有了许多不切实际的猜想。本来一桩小事，祗因他闲极生愁、庸人自扰，把嘉宇之言反覆品味了上千遍，心里暗示便不知强化了几分；以至他节后返校晚修时在校道遇着宛莹平安归来，竟痴痴地说不出一句话。

宛莹歇了三天，烧经已退了九成，心口也不大疼了；祗是不知同驰云闹成这般，何时才是了局；见他这般可气可怜，便主动设个台阶，走上前轻声问道：“嗳，你有什么要说的吗？”驰云就前日事故道了歉，又顺带为子帆求了情；弯来绕去，唯独不敢吐露心迹。他原想说，两人从小一直玩到大，比别人都要好，若是因为诗言子帆等外人而疏远了，不但可惜，也让人看笑话；祗是又嫌这话煽情，像极佟嘉宇的口吻，便生硬笑道：“今后可不可以不要拿诗言奚落我了？你同她根本不熟，无谓成日一直提她，弄得大家尴尬。”

宛莹听了这句，却解读成“你国中读的外校，与我早已生疏；此时你不但不了解诗言，也不了解我。我与她无论好或不好，至少已刻骨铭心；此刻还轮不到你一个外人置喙”，因又想道：“你终于承认，诗言是你永远的痛，那我以后可以不再开玩笑。只是但凡你心中还有一点友谊，也不会看我都这样了，还偏要提起她来。可见你心里装着诗言，就容不下其他人了；枉我还傻傻地把你看做个知己。”如此想着，便又灰了九分心，失落地问道：“好，我都听着了。你还有别的话要说么？”驰云真以为诗言之案经已了结；自己其它心迹又不便讲，因说：“除这件外也没了。”宛莹祗道他对自己早已无话可诉，难免心里阵阵酸楚、眼里簌簌掉泪，强忍着啜泣，哽咽道：“算我白认识了你。”说话间便碎步跑开。

驰云向来拙于人情世故，此刻当然不解宛莹心事；跑上前去把她拦住，却质问道：“这阵子又怎么了？我到底哪点惹到了你，明说出来岂不好？一番两次的，要哄到几时才算好？”宛莹哭道：“原来这叫哄我！既是这样，不如大家不要说话，彼此还能省些心。”驰云急得失控，赌气说：“如果我不烦你你才开心，也唯有敬而远之了。”宛莹怔了一怔，抬头凄怆道：“好，那就……”尚未说足半句，便又哽住了，于是掩了口继续前奔。

宛莹在前疾走，驰云在后尾随；两人追逐至温班廊下，不巧正遇着子帆与父亲杨老师在说话。原来开元节前叶堪双召走子帆后，又拉了杨父过来游说。杨父一介名教练，怎会看不出儿子在数竞上有天赋，只是未必能敌得过穆秀麟、孟驰云等人；本省数学奥林匹亚竞争又激烈，一等奖的名额统共就那么几个；要借数竞孤注一掷、以搏保送加分，自然太过冒险。然而这知行竞赛太新，他还搞不清是什么玩意，因而也没个成算。元旦假里，杨父本有一堆恭班的年度总结要写，这会再摊上了温班备课、数竞改卷，便彻底没了时间；放假三日，他竟在办公室电脑前坐了三日；好不容易趁着晚修前各班电视放映一周新闻的空档，才得跑去寻儿子商量；眼看晚修时间将至，正要回恭班巡堂，不料就碰上驰云两人。

杨父见他俩一个眼红声促、泪痕满面，一个脸色铁青、恨含眉间；便存心要调和调和。他与宛莹不熟，却刚刚代过奥数课，一心想跟驰云套近乎，便上前拍了拍驰云的肩，憨笑道：“孟驰云哪，怎么竟然对女同学发起脾气来啦？”子帆此刻尴尬万分，又生怕牵出自己撞人之过，遂埋怨道：“爸！人家两个吵架，你又乱凑什么热闹。”杨老师回过神来，一拍脑袋，仍心直口快地开玩笑道：“喔！有情况啊，哈哈！你看我这算什么眼神。从前总以为孟驰云找了个我们班的，我还专门在恭班附近偷偷地盯过一阵，想看看，唉，他找的是谁？眼光怎么样？明里私下却一次都没见他来过。我说奇怪……”一语未了，子帆忙把他推到楼梯间，催道：“你快回你的恭班值晚自习去。代了课还不过瘾，现在居然管起我们来了。骆岳文又不给你发代课奖金，你操这份闲心做什么。”宛莹听了杨父之语，更不知诗言之事到底闹得多大、有完没完，于是大受刺激。她自觉颜面尽失，早已夺门而入，伏于案上偷潸去了。

这边驰云也是一肚子气，不明白为何全世界都来打趣自己；眼看宛莹归座嘤泣，他生怕再惹众人口舌，于是只好悄悄地从后门兜进课室。原本两人闹不愉快，总预着官梅珩会来化解；然而今次吵得毫无退路，又一直未见梅珩归来，更不知这僵局如何打破。子帆虽将功补过，此刻奔忙两端、不懈安抚；孰料谁都不肯卖帐。他浪费了许多唇舌，中途再被杨父与叶教练叫走，终究无能为力。

驰云宛莹毕竟少不更事，各碰过一次壁，就都害怕再受打击；此间又无第三人介入转寰，他俩虽心中难安，却不知该怎样讲和。这边驰云想着，宛莹是女生、又初来乍到，自己理应让着她；唯独担心一句话讲错，又惹得她无名火起。那边宛莹也愿自己委屈些，唯要多一事不如少一事；祗是偏又快赶上月里那几天，前番受的气着的凉，这会都应在身上，成日痛得死去活来；即使有心应付驰云，也难控制住坏情绪。此时期末考试将近，她理科仍大面积欠账；不同驰云和好，就更无人帮她补习──官梅珩住院了，莫颖玟隔太远，其余女生同她一样自顾不暇，找别的男生又怕添了驰云的气；林韫之功课倒好，可请教过她几次后，她晚修便改习文科；听她室友说，她回寝后倒夜夜挑灯做理科卷子到凌晨一两点。

---

两人各有隐衷，以至直到寒假放假回家，谁也都没有理过谁。展眼除夕将近，驰云父母难得获批同时串休。当医生的最知争分夺秒；有此旧历年假，自然开车举家返乡、探候老人。医生假期固短，国高中生倒有教育部规定的三周寒假；然而西江教育署求好心切，为抓升学率，连国定假期也敢讨价还价、打个七折；一中再提前半周补课，竞赛组又抽两日集训，如此算下来驰云正月初五便要返校，倒正好与父母的上班日期相重。驰云在长身体，平日住宿就睡不够；这一路又舟车劳顿加拜年应酬；祗好每有闲暇就抓紧补觉，一个寒假竟白白睡了过去。

眼见又要开学，驰云便怏怏不乐，返程时空望着窗外，一路无话。等到驶过雪岭关，展眼春色夹道、梨蕊满野，其华似霰、其皎若莹，他心内才又泛上宛莹的名字，祗不知她此刻如何。驰云想着两人冷却了大半个月，再见时也理应冰释前嫌了；这才打起半分精神，倚着车窗，斟酌着重逢时的每句说话。待到出了果林，近了市肆，孟怀安把车开进服务区，寻了个排档就餐，他仍边吃边望着远山发呆。

驰云正嚼着饭，冷不防身后一个雄浑男声道：“哟！这不是孟医师么！”孟驰云拧头见是高秋黎，惊得差点跌出座位，心想他素日潜伏跟踪的功力再强，也无理由明察暗访到自己在这。听父母与他攀谈几句，才知他料理好家事后，不迟不早，偏也拣了这日乘长途客运返回西江。这会趁着中转休整，他赶紧下车抽烟解瘾，谁想就恰好就撞见驰云一家。驰云父母见到儿子班主任，哪有不客气奉承的；又要约吃饭，又邀他坐顺风车同走。高秋黎因盛情难却，竟真也应承下来；从大巴车取出了行李，便随驰云一家摆布。驰云讨厌他身上的残烟味，人情礼节也不顾，祗缩到后座角落里远远躲着。

高秋黎偏不放过驰云；与孟父神侃了半刻钟，便转身道：“孟驰云啊，听说我不在的这几天，温班状况很多咧。”驰云深知老师们故作此语，若非捉到了把柄，就是准备套口供。他以为高秋黎言指自己与宛莹，一时心悬嗓口，却装糊涂道：“老师讲的是官梅珩生病的事？”孟母也插话说：“是官议长的女儿罢？我们同事接诊过她几次，都说这孩子家教好，一点公主病都没，完全不像这种人家出身。”

高秋黎点头道：“这个官梅珩的确难得。虽然天资不是最出众，可是非常扎实低调，病得那么重才去请假，住了院还不忘复习功课。可另外有些人就不一样啦，”驰云以为他言指自己，便低头佯装系鞋带；再听他继续说道，“我刚回老家不久，有天早上五点多骆校长就打电话骂过来，说穆秀麟他们寝室深更半夜聚众打牌，聪明都不用在正经事上，净搞些歪门邪道。这件事我回去还要仔细追究──孟驰云啊，这学期高老师就不会再讲情面了咧！”驰云父母忙问自己儿子有无劣迹，又劳烦他严格管束；高秋黎却笑道：“孟驰云他们寝室倒好，不仅多才多艺，还很团结，”说着又重提一遍欧阳志夜哭之事。驰云未料到这番谈话也能跌宕起伏、峰回路转，至此才敢换个坐姿、长松口气。

驰云父母既为搏感情，也防打瞌睡；言笑不绝于路，与高秋黎相谈甚欢。不觉间就过了碧潭、总算在晚高峰前入了省城
