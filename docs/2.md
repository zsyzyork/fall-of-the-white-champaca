# 第二回 温良人概述温良俗，资优生初进资优班

转眼六七年又过，西江市立第一高中新校区早就竣工，如今已是落成启用的第四个年头。时值暑假，夏入三伏，酷热难当。空旷如也的校园里，仅有一父一女的身影。这对父女沿着大道，穿越广场；只见主楼横亘瑶琴山麓，楼高五层，玫瑰色的外墙，在正午日光下仿佛火烧云一般；在广场蒸腾的水气里，又如同蜃楼海市。

父女进了主楼，楼里又是另一个乾坤──迎面只见一个宽阔中庭，直通天窗，四周走马廊围绕，连同无数行空天梯，串联起各楼层厅室，纷呈缭乱。父女前后张望，一时竟找不到校长室的位置。半晌，只见水房走出一个阿姨，那父亲忙拦住打听骆校长在何处。那阿姨往右指着走廊尽头，说：“那边一直走到底，上二楼会看见有个小书房，校长今日就在那里。”

父女道了谢，便往阿姨所指方向寻去，二楼走廊的角落里头果然有个房间，小门微敞。轻叩几下门，不多时便有一个小麦肤色、神色威严的四十来岁中年人迎了出来，问道：“两位是白教授和白宛莹吧？我叫骆岳民。两位快快请进。”父女二人也问骆校长好。

这个访客原来是西江研究院的物理教授，名叫白璧合。这白璧合籍贯应天府，并非西江本地人。官礼朝在任市长时，大办教育、广纳贤才，白璧合便是那时留在了西江。其女宛莹今年十五，正是升高中的年龄。白宛莹乖巧伶俐，各门功课也优秀。只是她国小毕业那年，市府响应教育部号召，扬弃统一排名、择优录取的传统，制定出“统筹兼顾、就近就读”的升学原则，按学生的户籍住址，分派到临近的国中。此政一出，反响哗然。虽说有一起消息灵通的耳报神，早就走漏了风声；使得各路神通广大的家长，让孩子搞竞赛的搞竞赛、有才艺的练才艺、毫无特长的也趁早搬家，走各种偏门左道，就为进入明星国中；但更多的家长学生还是被打了个措手不及，白宛莹就是其中之一。白教授从未逼女儿学过提琴奥数。女儿艺考失败后，他倒是乐观豁达，想着少年失志，焉知非福；况且就近的国中虽不能与一中平起平坐，比起他处也已算是上乘。于是白璧合也不去走那些捷径，只教女儿不要灰心，安心读书；天无绝人之路，大不了高中再考回一中去。宛莹果然不负父望，中考不仅被西江一中高分录取，之后的分班测试更跻身全级前十。今日骆校长请白家父女前来，正为商讨宛莹的分班之事。

骆校长侧身让两人进入书房。宛莹只见当中一道锦绣围屏，把书房隔为里外两间──外间用作会客，并不藏书，唯有一面茶几、一长一短两张沙发、一个茶壶而已；想来屏风那边才是日常办公读书之处。骆校长虽不苟言笑，作风却毫不官派，此刻正忙着给两人倒水泡茶；未几，又接到里间传来的电话，说是市里有要紧事，遂请白家父女稍候片刻。

宛莹环顾四周装潢：不过是普通的石膏墙面、大理石地砖；与父亲的办公室并无二致。唯独左边墙上悬有一幅银蓝色的刺绣，图案是星辰大海，并无一个人物；粗看像葛式北斋的浮世绘，细看又有几分似梵谷的《星夜》。随视角左右变换，那银线绣的海涛居然也粼粼泛光，簇簇团团，颇有意趣。绣幅右下刺有“长风破浪”四个草书字，又有一行小字“西江一中九二届同学会敬赠新式粤绣”。右墙并无工艺品，墙角一盆大仙人掌，墙面一挂锦旗，鎏有“军民共建，鱼水情深”字样，是西江基地在新一中落成仪式上所赠。再看前边那一道屏风，工笔画的荔枝、龙眼栩栩如生，右上方是一个印，从右到左又有几行隶体字提款。白宛莹只看清“西江市立第一中学”八字，与方才进来校门上老省长的题字如出一辙。其余文字不及细看，骆校长就已从里间出来，在侧边的小沙发上坐下了，白教授父女这才敢坐。

骆校长先恭贺过宛莹，再对白父讲了几句俗话，便直奔主题道：“白宛莹很优秀。温、良、恭、俭四个班，准备选哪一个？”

“温班？”白教授怀疑自己没听真切。若莹报考一中前，他早做过研究：一中起初每级十二个班，按礼让廉耻忠孝仁爱信义和平的四维八德编号，只是耻字难听，耻班改叫知班。后来学校扩招，为让志向不同、水准参差的学生各有去处，又增添了礼乐射御四个术科班，以及温良恭俭四个资优班，其中又以温班最为特殊──别班的学生，需经中考考取，再测试排名，依次调配到良恭俭三班和其余班级；唯有温班的学生，本是一中国中部择优遴选、保送直升上来的，无需参与中考和分班试。因此，白璧合确认道，“校长说的，是直升班的那个温班？”

骆校长纠正道：“虽然很多人都有误解，但温班其实并非直升班。我校设立温班的目的，是在国中提前选拔出一批最顶尖的同学，在初三提前学习高中课程。这期间，每天都要考核，每周都有人被淘汰。即便是被淘汰下来参加中考的同学，绝大多数也有很强的实力。他们在分班试中名次靠前，组成了了良、恭、俭三个资优班的班底。不过，强中更有强中手。那些最出类拔萃的佼佼者，才有资格进入温班。白宛莹虽来自外校，没有经历这个磨练，成绩却大大超越了良恭俭三班的平均水准。这样的同学，温班自然也欢迎。”

白璧合笑道：“感谢骆校长的解惑。我们也只是有个好朋友孟大夫，其子就在如今的高一温班；才对此有所风闻，却未仔细打听，让校长见笑了。”

骆校长寻思并不识这个大夫，思索一会，沉吟道：“孟医生的公子，孟……可是今年全市奥数、物理竞赛双料一等奖的孟驰云？”白璧合连连道是：“就是这孩子。”骆校长道：“这个孟驰云，在温班也算是程度不错的了。为了不埋没这些同学，温班配备了全校最强的师资，希望三年后的大学学测，温班能出几个状元──怎么样，白教授和宛莹有何打算？”

白璧合迟疑一会，才说：“璧合由衷感激校长抬爱小女，破格批准入读温班。只不过听校长所言，温班人才济济。小女即使有幸进入，大概也只能陪居末位，反而给校长丢脸；倒不如在别班扎实念书为好。”

骆校长摆手道：“白教授请勿误会。一中行事规矩，绝无破格一说。温班原有五十六个名额，可是因为每年总有三四个同学国中毕业后要么选择转校，要么一意要去别的班级；所以这届只有五十二人。所以校方原则上也鼓励分班试前几名的同学进入温班就读。譬如这次的榜首，原本也是从温班淘汰之人。校方已再三给她机会，劝她回到温班。只是她决心要走，旁人勉强不来，只能由她。榜二榜三，校方也正在做他们的工作。再则，分班试前五十名里，只有白宛莹一人来自外校──不怕见笑，外校的师资，到底相对薄弱些──白宛莹能到达这个水准，足见天分、家教、毅力都非同凡响；若再有良师点拨，想必前途无量。退一步讲，即便不能适应，高一结业时还有文理分科的机会；若是另有打算，到时再商榷未迟。”

白璧合思忖片刻，大胆问道：“恕我冒昧──骆校长令媛品学兼优，全市皆知。听说和小女也是同届，想是也在温班了？”

骆校长毫不为意，从容答道：“不是我自谦，小女天资真的有限，潜力远不如人，国中就没考进资优班。还好她勤能补拙，这次分班试考得差强人意，能进哪个班就看她造化了。此外她国文写作上确实有点小聪明，温班的李老师令她中文课过去走读，我也是鼓励的。但无论如何，小女的实力也只够搏一搏中大南华这类大学；而像白宛莹一样的优秀生，原应有更高远的追求才是。”

白教授沉吟片刻，因问女儿意向如何。宛莹先听校长亲劝自己去温班，原有些虚荣自矜；再听班内竞争激烈，便有些吃惊畏难；后听旧友的名字，又有些动心好奇；终是踌躇难决。骆校长也不心急，仍让二人慢慢考虑。白璧合告辞回去，又找同事参谋。办公室同僚听闻后，莫不恭喜道贺，皆说“这班上多少学生出身权贵，以后都是西江呼风唤雨之人”、“多少富商想攀关系，花多少钱都不得进去”、“那铁面骆校长难得如此热诚，白兄好大面子，竟不领情”等语。去教育局开会，席间与马局长闲谈，对方也是拿一样的话来相劝。

殊不知白璧合为人父母，觉得这些功名利禄反属次要，最顾虑是温班学业苛重，生怕压垮爱女；最终还是深夜一人拎了些茶酒瓜果去造访孟驰云之父──好友孟槐安。那孟槐安却笑道：“别的倒罢，只是温班压力大是从哪里听来？别班学生遇到老师，犹如老鼠见猫；在直升班，竟然反了过来──老师每次上课都提心吊胆、畏首畏尾，生怕被刁难嘲笑。幸而这班学生也虽然淘气，内心其实都聪明乖觉、求好心切，没有不奋发向上的。”白璧合听了此言，方才放心。孟母在旁，又家长里短地说了些两家儿女如今在一个班，更应亲敬和睦等话，白教授便又坐了片刻方去。翌日，他把这些话转述给宛莹听。宛莹素来也是暗暗要强的的秉性，况且明年还有反悔机会，于是答应先念着温班再说。白父于是又禀谢校长、又宴请温班班主任与各任课老师，略去不提。

---

不久便是八月中旬。按西江一中惯例，高中部新生入学之前，须参加为期两周的统一国防教育与体能训练。温班诸生或有竞赛任务，本来得免；然而班主任高秋黎久闻这群学生桀骜不驯、自由放纵──这高秋黎据说以前也是个校长，对付学生自然很有一手；只是不知为何，竟甘愿到西江一中来当普通老师。今年接手了温班，高秋黎心想着，若不先立个下马威，将来这班学生恐怕更难钤束管教；便令本班男女生也随其余十九个班一起操练。即使是女生月事、或是体弱微恙之人──当然也包括一干取巧装病的，学前体训期间，高秋黎仍令每日走读来校、温习功课。

白宛莹早就听闻一中管束极多极严，诸如周一三五着校服正装、二四六穿校运动服，书包水杯不得悬挂、课本试卷必须收纳整齐；还有男生鬓角不得过耳、女生头发不得到肩，男女严禁并行独处，林林总总，不可胜记。入学前夕，她刚因为发禁，被迫剪掉留了三年的披肩长发，为此哭了半天。如今又要提早进校，她更百般不自在，装病拖延两日后，看是再也说不过去了，才不情不愿地被父亲送到高中部。

这天正是新生前往国防基地徒步行军的日子。白宛莹随父进了校门，走过运动场与科学楼，去女舍注册宿舍、沿途竟不见一人。直至从宿管办公室出来，才有个女生远远从右边走来。宛莹父女上前问了去高一温班的路。只见那女生眼如弯月、笑靥可亲，从容大方地说道：“是白叔叔和宛莹吧？我叫官颖梅，也是温班的。请随我这边走。”三人便穿过草坪，左转折过图书馆，上了四楼，才看见高一温、良两班，隔着楼下的小花园与俭、恭二班相望。白教授停在温班门口，絮絮地谢过颖梅一番、又叮咛了宛莹几句，这才放心离开。宛莹随那官颖梅进入班级，只见课室里有十余个女生参差坐着。

官颖梅拉着白宛莹的手，上了讲台，对台下笑道：“你们看谁来了！”随后拉她到人群里面，指着右边一个低头看书的单柔女生道：“这是班长容爱琳。”宛莹未见此人面目，却先闻到一阵香气。她心想，连班长上学都洒香水，难怪外人总说温班骄奢。正想着，班长款款站起，宛莹只见她眉目清秀。她柔声问了句好，再转向颖梅道“多谢你带她来”，便微笑不语。官颖梅笑道：“不用谢。我起先在餐厅看阿姨准备消暑糖水，回来刚巧就遇到了她。”容爱琳便问：“消暑糖水怎么了？”

宛莹见这班长举止文静、惜言省话，又并不像虚荣浅薄的人。正纳闷着，又听颖梅答道：“餐厅今早煮了绿豆薏米露。刚才把几十桶糖水移到外边放凉。现今天气湿热，那样露天放着，恐怕不出半天就要变质。浪费食物还事小，吃坏肚子事大。我就跟阿姨讲，不如把桶盖阖上，烧滚了放到水槽里，浇冷水降温。这样糖水不易变质，冷得还快。”

班长听了，道了一句“费心”，身后便传来一阵爽朗笑声：“果然是官小姐，到底比别人更懂养生。”官颖梅微红了脸，回头道：“才没有啦！”──又向宛莹介绍说：“这是望晓鹃，她的同桌是安若素，两位都是大才女；琴棋书画，无所不能。”宛莹只见后排坐了两人，左边五官精巧、柳眼梅腮的女子，被官颖梅夸赞为“才女”，脸颊登时红花飞泛，在座位上欠了欠身；右边女生长相英气，头发微卷；嘴里一边谦虚道“哪里哪里”，一边起身、一把握住宛莹的手，爽利地说：“欢迎你来，幸会！”宛莹也向两人问好。颖梅想起一件事，又问那站着的女生：“晓鹃，横幅厂商已联系好了，体训的标语想好了吗。”那女生笑道：“早已写好放你桌上了，一共三个文案，任你挑选。”

官、白两人又转到相邻一组，该组第二行是颖梅的座位，桌面上果然压了一页纸，纸上密密麻麻地写着蓝色水笔字，字迹工整却有些男孩子气。颖梅收起纸张，继续领着宛莹去别组打招呼。诸多女生中，有姓名生僻不会念的，也有起名跟风、导致重名的；有相貌出众好认的，也有相貌一时难辨的──毕竟大家都剪了短发，撞脸之人无法靠发型区别。独有一人并不剪发，只绾起了一头青丝，悄悄将发尾收拢内折，粗看过去竟与短发无异。这机灵做法让宛莹印象深刻，所以记得此人叫林韫之。

颖梅介绍了一圈，又带宛莹回到前排，指着林韫之正前方的位子、以及隔壁组第一排的两个空位，问宛莹说：“你要选哪里坐？”林韫之此时站起来，悄悄打量着白宛莹与自己身量相仿，便凑到她耳边笑着说：“我告诉你，不出两个礼拜，咱们的课本习题就堆不下了。我左前边有人坐了，你坐旁边那组，你两人都多个抽屉。”宛莹因问前排怎会有座位空着，颖梅抚着宛莹的肩，对视笑道：“还不是远方有贵客来，老师早就把座位预留好了；我们碰都不敢碰呢！现在贵客终于来了，‘蓬门今始为君开’！”又补充说，“除此以外，以后骆校长的女儿有时也会过来上中文课。你们俩一来，咱们班就更热闹了。”

众女生见过了宛莹，仍返回坐位各做各事。颖梅怕冷落了她，便坐到她身旁空位相陪，如对校园场地有所不熟、或是学习生活遇了困难、再或被男生捉弄欺负，让她告诉班长或自己就好。宛莹一刻钟前还在悒郁，看颖梅如此亲切诚恳，不禁心窝一暖、眼眶一酸。余人都不理论，唯有望晓鹃是个热心外向爱倾谈的，此刻也凑过来玩笑打趣说：“官小姐可真是宠你了。我跟她同学三年，也没听她说要罩着让我不受委屈。可见我们是高攀不起的，你倒是快认她做姐姐吧。”颖梅忙搂住晓鹃，捏着她脸皮笑道：“谁敢欺负你？这么多年你不戏弄别人，我们已经谢天谢地了。”

宛莹因问：“人人都说咱们班同学知书达理，竟也有欺负别人的？”不等颖梅开口，晓鹃嗤地一声冷笑道：“等会你就知道了！咱们班的男生，一个比一个难缠。尤其是数学组里的那几个，像穆秀临、杨子帆的为人；小聪明没处使，一日不生事就闲得慌。还有那个孟驰云，最是恃才傲物、不积口德的。”宛莹望向众人，虽无人答言，那林韫之和安若素都抿唇而笑；官颖梅笑道：“男生多少难免幼稚不懂事，哪里就成了恶人了。劝你少说两句罢，别吓着新同学了。”晓鹃道：“我没说是恶人。正常起来也很正常，只是使坏时没个分寸。”

宛莹不知如何回应，想了一会说：“听说温班班主任捉起人来毫不手软，其他老师也都很严。他们管教一阵，说不定就好了呢。”晓鹃叹道：“希望吧，反正不要像国中部那些老师一样，一个个全都欺软怕硬，功课好的学生违纪，他们从来只是装装样子；生怕说了几句重话，他们期末赌气考砸，自己年终奖金就没了。这些家伙就是凭着老师的娇纵，所以才敢这么肆无忌惮……”

一语未了，只听见隐约的哨声呼号声，想是大部队已经体训回来；望晓鹃便压低声音道，“罢了罢了，这种时候说人坏话太危险。”颖梅呵呵笑道：“你也知道说的是坏话。”众人便都散去。

略坐了一阵，忽听得一阵喧闹，宛莹猜是体训的队伍解散了。果然，那笑闹声不一会就由远及近。有几个跑得快的，已争先冲入课室。留神到宛莹的人，都又多看了几眼；有向她问好的，也有不问好的。

忽然，后窗外一个惊呼声起：“哇，新来的是个美女。正点喔！”宛莹自然不好意思转头去看聒噪者是谁，只听另一人低声反问：“你不是说林韫之最漂亮么。”那惊呼之人便毫不掩饰地品评道：“废话！若是身材再好点，那就完美了。”宛莹心恼谁人如此轻薄嚣张，到底悄悄地瞄了一眼；只见说话两人皆肤色黝黑，一个正咧嘴傻笑，另一个倒是狂狷的样子，心想这必是穆、杨二人了。

惊魂甫定，不知哪里又传来一阵打闹声，忽见一个白皙俊秀、大眼长睫的男生停在自己身前，邪魅笑道：“新同学刚来吗？现在天热小心中暑，我有把阳伞送你。你嫌弃的话也可以扔掉。”她正疑惑为何有人无故献殷勤，身后便有女生哀叫道：“穆秀临我求你别闹了，把伞还给我罢！”宛莹才知道这个男生才是晓鹃眼中钉之一。转眼间那穆秀临于又提伞张扬而去，途中不忘弄乱这人的头发，摘掉那人的帽子。一时间人声鼎沸，鸡飞狗跳。官颖梅早已离座去调停安抚，只是刚压住这边，那边又闹了起来。班长倒是很自若地一边写作业，一边唤着“安静”；只是嗓门太轻，完全没人搭理。

白宛莹这才知晓鹃之言果然没差。自己孤孑一身，今后还不知将如何担惊受怕，未免一阵难过。还在伤神，却见一个剑眉风目的男生从前门大步流星地走近──这不是孟驰云是谁。宛莹正准备看他又要如何兴风作浪──谁料他昂首阔步、目不斜视、一人不理，飞也似地倏忽而过，当自己是透明人一般──宛莹见状，心又不禁一冷。或许是自己剪短了头发，他一时没认出来呢。宛莹如此想着，却越想越难过。

众人闹了不知多久，有人忽喊“班主任来了！”霎时间，课室鸦雀无声。宛莹一回头，果真看见班主任高秋黎正背着手，静静从后门踱进来。只见他不仅毫无愠色，反而笑意盈盈，扫视一周，悠悠说道：“刚才很远就听到有打闹声，我就在跟骆校长开玩笑：谁说现在的中学生都是草莓族、徒步十公里吃不消的。你看我们温班的同学，一点都不觉得累嘛！”说完，依然脸挂笑容，又缓缓望了台下一圈。所有人都屏息不敢说话，都等着看他葫芦里卖什么药。他卖够了关子，才说“今早你们杨老师出了一道很好的数学题。穆秀临，杨子帆，还有那谁来着，你们都很有精神。等一下干脆午饭也别吃了，到我办公室去，大家一起研究研究数学，消耗一下体力。怎么样啊，杨子帆？看看你爸爸出的题你做不做得出来？”台下众人闻言，都已噤若寒蝉。

此时刚好放学铃响，高秋黎仍不紧不慢说：“我知道你们赶着吃饭，所以只讲一句──”诸人早就作好起跑姿势，焦急地待他讲完，随时准备冲向餐厅──却听他道：“今天，白宛莹同学加入了咱们班，我们请白宛莹同学自我介绍！”这“一句话”后，顿时怨声四起。宛莹心想这老师怎么净给自己招祸，无奈三言两语混了过去；谁知那高秋黎又接过话茬，接连补充了好几句，再留下先前打闹的三四个人，方肯放余人离去。宛莹正要走时，却也被他截了下来，单独嘱咐一番。宛莹只听他颠来倒去一大串话，什么“脚踏实地”、什么“艰苦奋斗”、什么“磨练心性”，终究不知所云。等听他训完话，去到餐厅，大多数人早已用完午餐，打饭阿姨开始把各盘残羹冷炙归并到一起卖，观之毫无食欲；颖梅说的糖水也早已告罄。宛莹再回头去宿舍管理处报到，偏又过了午休时间，宿管员工都在各处查房不得闲。如此一来，宛莹唯有悻悻然地走回课室。她正路过男生宿舍前的草坪，行道树后却冷不防地钻出一个人来。她定睛一看──
